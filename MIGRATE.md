# 迁移

请参阅以下关于迁移 LangChain 代码的指南：

* 迁移到 [LangChain v0.3](https://python.langchain.com/docs/versions/v0_3/)
* 迁移到 [LangChain v0.2](https://python.langchain.com/docs/versions/v0_2/)
* 从 [LangChain 0.0.x Chains](https://python.langchain.com/docs/versions/migrating_chains/) 迁移
* 升级到 [LangGraph Memory](https://python.langchain.com/docs/versions/migrating_memory/)

[LangChain CLI](https://python.langchain.com/docs/versions/v0_3/#migrate-using-langchain-cli) 可以帮助您自动升级代码以使用非弃用的导入。如果您仍在使用 LangChain 的 0.0.x 或 0.1.x 版本，这将特别有用。
