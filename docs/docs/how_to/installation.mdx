# 如何安装 LangChain 包

LangChain 生态系统被拆分为不同的包，这允许您精确选择要安装的功能。

## 官方发布版

要安装主要的 `langchain` 包，请运行：

import TabItem from '@theme/TabItem';

  <TabItem value="pip" label="Pip" default>

虽然此包是使用 LangChain 的一个合理起点，但 LangChain 的许多价值在于将其与各种模型提供商、数据存储等集成。默认情况下，执行此操作所需的依赖项不会安装。您需要单独安装特定集成所需的依赖项，如下所示。

## 生态系统包

除了 `langsmith` SDK 之外，LangChain 生态系统中的所有包都依赖于 `langchain-core`，它包含其他包使用的基类和抽象。下面的依赖关系图显示了不同包之间的关系。有向箭头表示源包依赖于目标包：

![](/img/ecosystem_packages.png)

安装包时，您无需显式安装该包的显式依赖项（例如 `langchain-core`）。但是，如果您使用的是仅在特定版本的该依赖项中提供的功能，则可以选择这样做。如果这样做，您应该确保已安装或固定的版本与您使用的任何其他集成包兼容。

### LangChain Core
`langchain-core` 包包含 LangChain 生态系统其余部分使用的基本抽象，以及 LangChain Expression Language。它由 `langchain` 自动安装，但也可以单独使用。使用以下命令安装：

```bash
pip install langchain-core
```

### 集成包

某些集成（如 OpenAI 和 Anthropic）有自己的包。任何需要自己包的集成将在[集成文档](/docs/integrations/providers/)中进行记录。您可以在[API 参考](https://python.langchain.com/api_reference/)的“合作伙伴库”下拉菜单下找到所有集成包的列表。要安装其中一个包，请运行：

```bash
pip install langchain-openai
```

尚未拆分为自己包的任何集成将保留在 `langchain-community` 包中。使用以下命令安装：

```bash
pip install langchain-community
```

### LangChain Experimental
`langchain-experimental` 包包含实验性的 LangChain 代码，用于研究和实验用途。使用以下命令安装：

```bash
pip install langchain-experimental
```

### LangGraph
`langgraph` 是一个用于构建具有 LLM 的有状态、多参与者应用程序的库。它与 LangChain 无缝集成，但也可以独立使用。使用以下命令安装：

```bash
pip install langgraph
```

### LangServe
LangServe 帮助开发人员将 LangChain 可运行对象和链部署为 REST API。LangServe 由 LangChain CLI 自动安装。如果未使用 LangChain CLI，请使用以下命令安装：

```bash
pip install "langserve[all]"
```

以同时安装客户端和服务器依赖项。或者，使用 `pip install "langserve[client]"` 安装客户端代码，使用 `pip install "langserve[server]"` 安装服务器代码。

### LangChain CLI
LangChain CLI 对于使用 LangChain 模板和其他 LangServe 项目非常有用。使用以下命令安装：

```bash
pip install langchain-cli
```

### LangSmith SDK
LangSmith SDK 由 LangChain 自动安装。但是，它不依赖于 `langchain-core`，如果需要，可以独立安装和使用。如果您不使用 LangChain，可以使用以下命令安装：

```bash
pip install langsmith
```

### 从源代码安装

如果您想从源代码安装包，可以通过克隆[主要的 LangChain 存储库](https://github.com/langchain-ai/langchain)，进入您想安装的包的目录 `PATH/TO/REPO/langchain/libs/{package}`，然后运行：

```bash
pip install -e .
```

LangGraph、LangSmith SDK 和某些集成包位于主要的 LangChain 存储库之外。您可以在[此处](https://github.com/langchain-ai)查看所有存储库。
