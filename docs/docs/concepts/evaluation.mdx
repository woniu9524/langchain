# 评估
<span data-heading-keywords="evaluation,evaluate"></span>

评估是衡量 LLM 驱动的应用程序性能和有效性的过程。
它涉及根据一组预定义的标准或基准测试模型的响应，以确保其达到期望的质量标准并实现预期目的。
此过程对于构建可靠的应用程序至关重要。

![](/img/langsmith_evaluate.png)

[LangSmith](https://docs.smith.langchain.com/) 在此过程中提供了一些帮助：

- 通过其跟踪和注释功能，可以更轻松地创建和整理数据集
- 它提供了一个评估框架，帮助您定义指标并在数据集上运行您的应用程序
- 它允许您跟踪随时间推移的结果，并自动按计划或作为 CI/Code 的一部分运行评估器

要了解更多信息，请查看[此 LangSmith 指南](https://docs.smith.langchain.com/concepts/evaluation)。
