## 概述

一种常见的用于提高性能的提示技术是将示例作为提示的一部分。这被称为[少样本提示](/docs/concepts/few_shot_prompting)。

这为[语言模型](/docs/concepts/chat_models/)提供了它应该如何表现的具体示例。
有时这些示例是硬编码到提示中的，但对于更高级的情况，动态选择它们可能会更好。

**示例选择器**是负责选择然后将示例格式化为提示的类。
