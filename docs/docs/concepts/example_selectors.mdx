# 示例选择器

:::note 先决条件

- [聊天模型](/docs/concepts/chat_models/)
- [少样本提示](/docs/concepts/few_shot_prompting/)
:::

## 概述

一种常见的技术，旨在通过在提示中包含示例来获得更好的性能。这就是所谓的[少样本提示](/docs/concepts/few_shot_prompting)。

这为[语言模型](/docs/concepts/chat_models/)提供了具体示例，说明它应该如何表现。
有时这些示例会被硬编码到提示中，但在更高级的情况下，动态选择它们可能会更方便。

**示例选择器** 是负责选择然后将示例格式化为提示的类。
