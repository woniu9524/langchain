# 输入字符串，输出字符串的 LLM

:::tip
您可能正在寻找 [Chat Model Concept Guide](/docs/concepts/chat_models) 页面以获取更多信息。
:::

LangChain 为接受字符串作为输入并返回字符串作为输出的旧语言模型提供了实现。这些模型通常命名时不带“Chat”前缀（例如 `Ollama`、`Anthropic`、`OpenAI` 等），并且可能包含“LLM”后缀（例如 `OllamaLLM`、`AnthropicLLM`、`OpenAILLM` 等）。这些模型实现了 [BaseLLM](https://python.langchain.com/api_reference/core/language_models/langchain_core.language_models.llms.BaseLLM.html#langchain_core.language_models.llms.BaseLLM) 接口。

用户几乎应该只使用较新的 [Chat Models](/docs/concepts/chat_models)，因为大多数模型提供商都采用了类似聊天的接口来与语言模型进行交互。
