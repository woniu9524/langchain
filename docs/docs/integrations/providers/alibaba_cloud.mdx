# 阿里云

>[阿里巴巴集团控股有限公司 (Wikipedia)](https://en.wikipedia.org/wiki/Alibaba_Group) (英文名 `Alibaba`)
> (中文: 阿里巴巴) 是一家中国的跨国科技公司，专注于电子商务、零售、互联网和技术。
> 
> [阿里云 (Wikipedia)](https://en.wikipedia.org/wiki/Alibaba_Cloud) (英文名 `Aliyun`，中文: 阿里云；汉语拼音: Ālǐyún；字面意思: 'Ali Cloud') 是一家云计算公司，是`阿里巴巴集团`的子公司。`阿里云`为在线业务及阿里巴巴自身的电子商务生态系统提供云计算服务。
