# DocArray

> [DocArray](https://docarray.jina.ai/) 是一个用于嵌套、非结构化、多模态数据传输的库，包括文本、图像、音频、视频、3D网格等。它允许深度学习工程师使用 Pythonic API 高效地处理、嵌入、搜索、推荐、存储和传输多模态数据。

## 安装和设置

我们需要安装 `docarray` Python 包。

```bash
pip install docarray
```

## 向量存储

LangChain 提供了对 `DocArray` 库中 `内存中` 和 `HNSW` 向量存储的访问。

请参阅 [用法示例](/docs/integrations/vectorstores/docarray_hnsw)。

```python
from langchain_community.vectorstores import DocArrayHnswSearch
```
请参阅 [用法示例](/docs/integrations/vectorstores/docarray_in_memory)。

```python
from langchain_community.vectorstores import DocArrayInMemorySearch
```
