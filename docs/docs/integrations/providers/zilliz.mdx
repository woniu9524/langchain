# Zilliz

>[Zilliz Cloud](https://zilliz.com/doc/quick_start) 是 `LF AI Milvus®` 的全托管云服务。


## 安装与设置

安装 Python SDK：
```bash
pip install pymilvus
```

## Vectorstore

一个围绕 Zilliz 索引的封装库，可让你将其用作 vectorstore，
无论是用于语义搜索还是示例选择。

```python
from langchain_community.vectorstores import Milvus
```

有关 Milvus 封装库的更详细的教程，请参阅 [此笔记本](/docs/integrations/vectorstores/zilliz)
