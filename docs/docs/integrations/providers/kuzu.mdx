# Kùzu

> [Kùzu](https://kuzudb.com/) 是一个可嵌入的、可扩展的、极其快速的图数据库。
> 它拥有宽松的 MIT 许可证，您可以在[此处](https://github.com/kuzudb/kuzu)查看其源代码。

> Kùzu 的主要特点：
>- 性能和可扩展性：实现了现代、最先进的图连接算法。
>- 易用性：设置和入门非常容易，因为没有服务器（而是嵌入式架构）。
>- 互操作性：可以方便地从外部列格式、CSV、JSON 和关系数据库扫描和复制数据。
>- 结构化属性图模型：实现了属性图模型，并增加了结构。
>- Cypher 支持：允许使用声明式查询语言 Cypher 方便地查询图。

> 通过访问 [文档](https://docs.kuzudb.com/) 来开始使用 Kùzu。

## 安装和设置

如下安装 Python SDK：

```bash
pip install -U langchain-kuzu
```
