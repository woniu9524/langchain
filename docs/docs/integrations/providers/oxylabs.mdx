# Oxylabs

[Oxylabs](https://oxylabs.io/) 是一个市场领先的网络情报收集平台，以最高的商业、道德和合规标准为驱动，使全球企业能够解锁数据驱动的洞察。

[langchain-oxylabs](https://pypi.org/project/langchain-oxylabs/) 实现了使 LLM 与 Oxylabs Web Scraper API 交互的工具。


## 安装和设置

```bash
pip install langchain-oxylabs
```
