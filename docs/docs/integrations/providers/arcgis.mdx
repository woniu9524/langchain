# ArcGIS

>[ArcGIS](https://www.esri.com/en-us/arcgis/about-arcgis/overview) 是由 [Esri](https://www.esri.com/) 开发和维护的客户端、服务器和在线地理信息系统软件系列。
>
>`ArcGISLoader` 使用 `arcgis` 包。
> `arcgis` 是一个 Python 库，用于矢量和栅格分析、地理编码、地图制作、路径规划和导航。它管理、组织和维护您的 GIS 中的用户、组和信息项。
>它能够访问来自 `Esri` 和其他权威来源的现成地图和精选地理数据，并且还可以与您自己的数据一起使用。

## 安装与设置

我们需要安装 `arcgis` 包。

```bash
pip install -U arcgis
```

## 文档加载器
