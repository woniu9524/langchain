# Dappier

[Dappier](https://dappier.com) 将任何 LLM 或您的 Agentic AI 连接到来自受信任来源的实时、已获授权的专有数据，使您的 AI 成为任何领域的专家。我们的专业模型包括实时网络搜索、新闻、体育、金融股市数据、加密货币数据以及来自高级出版商的独家内容。请访问 [marketplace.dappier.com](https://marketplace.dappier.com) 探索我们市场中的各种数据模型。

[Dappier](https://dappier.com) 提供经过丰富化、即时可用且上下文相关的字符串数据，针对与 LangChain 的无缝集成进行了优化。无论您是构建对话式 AI、推荐引擎还是智能搜索，Dappier 的 LLM-agnostic RAG 模型都能确保您的 AI 能够访问经过验证的最新数据——无需构建和管理自己的检索管道的复杂性。

## 安装和设置

安装 ``langchain-dappier`` 并设置环境变量 ``DAPPIER_API_KEY``。

```bash
pip install -U langchain-dappier
export DAPPIER_API_KEY="your-api-key"
```

我们还需要设置 Dappier API 凭证，这些凭证可以在 [Dappier 网站](https://platform.dappier.com/profile/api-keys) 上生成。

我们可以通过访问 [Dappier 市场](https://platform.dappier.com/marketplace) 来查找支持的数据模型。
