# Notion 数据库

>[Notion](https://www.notion.so/) 是一个协作平台，支持修改后的 Markdown，集成了看板、任务、维基和数据库。它是一个用于笔记记录、知识和数据管理以及项目和任务管理的一体化工作空间。

所有说明都在下面的示例中。

from langchain_community.document_loaders import NotionDirectoryLoader, NotionDBLoader
