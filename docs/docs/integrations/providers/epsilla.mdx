# Epsilla

本页介绍如何在 LangChain 中使用 [Epsilla](https://github.com/epsilla-cloud/vectordb)。
本页分为两部分：安装和设置，以及特定 Epsilla 封装器的参考。

## 安装和设置

- 使用 `pip/pip3 install pyepsilla` 安装 Python SDK

## 封装器

### VectorStore

Epsilla 向量数据库提供了一个封装器，允许您将其用作向量存储，无论是用于语义搜索还是示例选择。

导入此向量存储：

```python
from langchain_community.vectorstores import Epsilla
```

有关 Epsilla 封装器的更详细教程，请参阅 [此 notebook](/docs/integrations/vectorstores/epsilla)
