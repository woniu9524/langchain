# Yandex

与 Yandex Cloud 相关的所有功能

>[Yandex Cloud](https://cloud.yandex.com/en/) 是一个公共云平台。

## 安装和设置

可以通过 pip 从 PyPI 安装 Yandex Cloud SDK：

```bash
pip install yandexcloud
```

## LLMs

### YandexGPT

请参阅[使用示例](/docs/integrations/llms/yandex)。

```python
from langchain_community.llms import YandexGPT
```

## Chat models

### YandexGPT

请参阅[使用示例](/docs/integrations/chat/yandex)。

```python
from langchain_community.chat_models import ChatYandexGPT
```

## Embedding models

### YandexGPT

请参阅[使用示例](/docs/integrations/text_embedding/yandex)。

```python
from langchain_community.embeddings import YandexGPTEmbeddings
```

## Parser

### YandexSTTParser

它用于转录和解析音频文件。

`YandexSTTParser` 与 `OpenAIWhisperParser` 类似。
请参阅[与 OpenAIWhisperParser 的使用示例](/docs/integrations/document_loaders/youtube_audio)。

```python
from langchain_community.document_loaders import YandexSTTParser
```
