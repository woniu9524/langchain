# Postgres Embedding

> [pg_embedding](https://github.com/neondatabase/pg_embedding) 是一个开源包，用于
> 向量相似性搜索，它使用 `Postgres` 和 `分层可导航小世界`
> 算法进行近邻搜索。

## 安装与设置

我们需要安装几个 Python 包。

```bash
pip install psycopg2-binary
```

## Vector Store

请参阅 [使用示例](/docs/integrations/vectorstores/pgembedding)。

```python
from langchain_community.vectorstores import PGEmbedding
```
