# Postgres Embedding

> [pg_embedding](https://github.com/neondatabase/pg_embedding) 是一个开源包，用于使用 `Postgres` 和 `分层可导航小世界` (Hierarchical Navigable Small Worlds) 算法进行向量相似性搜索，以实现近似最近邻搜索。

## 安装和设置

我们需要安装几个 Python 包。

```bash
pip install psycopg2-binary
```

## 向量存储

请参阅 [用法示例](/docs/integrations/vectorstores/pgembedding)。

```python
from langchain_community.vectorstores import PGEmbedding
