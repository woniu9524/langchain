# AgentQL

[AgentQL](https://www.agentql.com/) 通过 [AgentQL 查询](https://docs.agentql.com/agentql-query) 或自然语言提示，提供网页交互和从任何网页中提取结构化数据。AgentQL 可用于多种语言和网页，并且随着时间的推移和变化不会被破坏。

## 安装和设置

安装集成包：

```bash
pip install langchain-agentql
```

## API Key

从我们的 [开发者门户](https://dev.agentql.com/) 获取 API Key，并将其添加到您的环境变量中：
```
export AGENTQL_API_KEY="your-api-key-here"
```

## DocumentLoader
AgentQL 的文档加载器通过 AgentQL 查询从任何网页提取结构化数据。

```python
from langchain_agentql.document_loaders import AgentQLLoader
```
请参阅我们的[文档加载器文档和用法示例](/docs/integrations/document_loaders/agentql)。

## Tools 和 Toolkits
AgentQL 工具通过 AgentQL 查询或自然语言提示，提供网页交互和从任何网页中提取结构化数据。

```python
from langchain_agentql.tools import ExtractWebDataTool, ExtractWebDataBrowserTool, GetWebElementBrowserTool
from langchain_agentql import AgentQLBrowserToolkit
```
请参阅我们的[工具文档和用法示例](/docs/integrations/tools/agentql)。
