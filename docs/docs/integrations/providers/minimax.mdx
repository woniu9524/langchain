# Minimax

>[Minimax](https://api.minimax.chat) 是一家中国初创公司，为企业和个人提供自然语言处理模型。

## 安装与设置
获取一个 [Minimax API 密钥](https://api.minimax.chat/user-center/basic-information/interface-key) 并将其设置为环境变量 (`MINIMAX_API_KEY`)
获取一个 [Minimax 组 ID](https://api.minimax.chat/user-center/basic-information) 并将其设置为环境变量 (`MINIMAX_GROUP_ID`)


## LLM

提供了一个 Minimax LLM 包装器，您可以通过以下方式访问：
查看 [用法示例](/docs/integrations/llms/minimax)。

```python
from langchain_community.llms import Minimax
```

## Chat Models

查看 [用法示例](/docs/integrations/chat/minimax)

```python
from langchain_community.chat_models import MiniMaxChat
```

## Text Embedding Model

提供了一个 Minimax Embedding 模型，您可以通过以下方式访问：
```python
from langchain_community.embeddings import MiniMaxEmbeddings
```
