# Graph RAG

## 概述

[Graph RAG](https://datastax.github.io/graph-rag/) 提供了一个检索器接口，它结合了向量上**非结构化**的相似性搜索和元数据属性的**结构化**遍历。这使得在**现有**向量存储上实现基于图的检索。

## 安装和设置

```bash
pip install langchain-graph-retriever
```

## 检索器

```python
from langchain_graph_retriever import GraphRetriever
```

更多信息，请参阅 [Graph RAG 集成指南](/docs/integrations/retrievers/graph_rag)。
