# AskNews

[AskNews](https://asknews.app/) 通过每日处理和索引超过 300,000 篇文章，为语言模型提供最新的全球或历史新闻，通过低延迟的端点提供经过提示优化的响应，并确保新闻报道的透明度和多样性。

## 安装和设置

首先，您需要安装 `asknews` Python 包。

```bash
pip install asknews
```

您还需要设置我们的 AskNews API 凭据，这些凭据可以在 [AskNews 控制台](https://my.asknews.app/) 生成。
