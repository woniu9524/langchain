# Runhouse

本页面介绍了如何在 LangChain 中使用 [Runhouse](https://github.com/run-house/runhouse) 生态系统。
内容分为三个部分：安装与设置、LLM 和 Embeddings。

## 安装与设置
- 使用 `pip install runhouse` 安装 Python SDK
- 如果你想使用按需集群，请使用 `sky check` 检查你的云凭证

## 自托管 LLM
对于一个基础的自托管 LLM，你可以使用 `SelfHostedHuggingFaceLLM` 类。对于更
定制化的 LLM，你可以使用 `SelfHostedPipeline` 父类。

```python
from langchain_community.llms import SelfHostedPipeline, SelfHostedHuggingFaceLLM
```

关于自托管 LLM 的更详细演练，请参阅[这篇 notebook](/docs/integrations/llms/runhouse)。

## 自托管 Embeddings
通过 Runhouse，有多种方式在 LangChain 中使用自托管 embeddings。

对于一个来自 Hugging Face Transformers 模型的基础自托管 embedding，你可以使用
`SelfHostedEmbedding` 类。
```python
from langchain_community.llms import SelfHostedPipeline, SelfHostedHuggingFaceLLM
```

关于自托管 Embeddings 的更详细演练，请参阅[这篇 notebook](/docs/integrations/text_embedding/self-hosted)。
