# FireCrawl

>[FireCrawl](https://firecrawl.dev/?ref=langchain) 可以抓取并转换任何网站，生成 LLM 就绪的数据。
> 它会抓取所有可访问的子页面，并为每个页面提供干净的 Markdown 和元数据。无需站点地图。


## 安装和设置

安装 Python SDK：

```bash
pip install firecrawl-py==0.0.20
```

## 文档加载器

请参阅[用法示例](/docs/integrations/document_loaders/firecrawl)。

```python
from langchain_community.document_loaders import FireCrawlLoader
