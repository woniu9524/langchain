# MariTalk

>[MariTalk](https://www.maritaca.ai/en) 是一个基于 LLM 的聊天机器人，旨在满足巴西的需求。

## 安装与设置

您需要获取 MariTalk API 密钥。

您还需要安装 `httpx` Python 包。

```bash
pip install httpx
```

## Chat 模型

请参见[使用示例](/docs/integrations/chat/maritalk)。

```python
from langchain_community.chat_models import ChatMaritalk
```
