# Yahoo

>[Yahoo (Wikipedia)](https://en.wikipedia.org/wiki/Yahoo) 是美国一家网络服务提供商。
>
> 它提供了一个门户网站、搜索引擎 Yahoo Search 以及相关服务，包括 `My Yahoo`、`Yahoo Mail`、`Yahoo News`、`Yahoo Finance`、`Yahoo Sports` 及其广告平台 `Yahoo Native`。
