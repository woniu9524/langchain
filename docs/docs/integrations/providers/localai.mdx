# LocalAI

>[LocalAI](https://localai.io/) 是免费的、开源的 OpenAI 替代方案。
> `LocalAI` 可作为兼容 OpenAI API 规范的、即插即用的 REST API，用于本地推理。它允许您使用消费级硬件在本地或本地部署运行大型语言模型、生成图像、音频（以及更多）等，支持多种模型系列和架构。

:::caution
为确保兼容性，请确保您使用的 `openai` SDK 版本为 **0.x**。
:::

:::info
`langchain-localai` 是 LocalAI 的第三方集成包。它提供了一种在 Langchain 中使用 LocalAI 服务的方式。
源代码可在 [Github](https://github.com/mkhludnev/langchain-localai) 上找到。
:::

## 安装和设置

我们需要安装几个 Python 包：

```bash
pip install tenacity openai
```
