# Weights & Biases

>[Weights & Biases](https://wandb.ai/) 是一个AI开发者平台提供商，用于训练和微调AI模型以及开发AI应用程序。
 
`Weights & Biases` 产品可用于在训练期间记录指标和构件，并跟踪代码的执行。

在 LangChain 中使用 `Weights & Biases` 产品有几种主要方式：
- 使用 `wandb_tracing_enabled`
- 使用 `Weave` 轻量级工具包
- 使用 `WandbCallbackHandler` (已弃用)


## wandb_tracing_enabled

请参阅 [使用示例](/docs/integrations/providers/wandb_tracing)。

请参阅 [W&B 文档](https://docs.wandb.ai/guides/integrations/langchain)。

```python
from langchain_community.callbacks import wandb_tracing_enabled
```
