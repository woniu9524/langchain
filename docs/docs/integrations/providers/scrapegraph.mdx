# ScrapeGraph AI

>[ScrapeGraph AI](https://scrapegraphai.com) 是一项提供 AI 驱动的网络抓取功能的.*
>它提供用于提取结构化数据、将网页转换为 Markdown 以及使用自然语言提示处理本地 HTML 内容的工具。

## 安装和设置

安装所需的包：

```bash
pip install langchain-scrapegraph
```

设置您的 API 密钥：

```bash
export SGAI_API_KEY="your-scrapegraph-api-key"
```

## 工具

请参阅[用法示例](/docs/integrations/tools/scrapegraph)。

有四种可用工具：

```python
from langchain_scrapegraph.tools import (
    SmartScraperTool,    # 从网站提取结构化数据
    MarkdownifyTool,     # 将网页转换为 Markdown
    LocalScraperTool,    # 处理本地 HTML 内容
    GetCreditsTool,      # 检查剩余 API 积分
)
```

每种工具都有特定的用途：

- `SmartScraperTool`: 从给定 URL、提示和可选输出模式的网站提取结构化数据
- `MarkdownifyTool`: 将任何网页转换为干净的 Markdown 格式
- `LocalScraperTool`: 从给定提示和可选输出模式的本地 HTML 文件提取结构化数据
- `GetCreditsTool`: 检查您剩余的 ScrapeGraph AI 积分
