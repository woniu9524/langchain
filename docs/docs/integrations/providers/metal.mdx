# Metal

本页面介绍如何在 LangChain 中使用 [Metal](https://getmetal.io)。

## 什么是 Metal？

Metal 是一个为生产环境构建的托管检索与内存平台。您可以轻松地将数据索引到 `Metal` 中，并对其进行语义搜索和检索。

![Metal 仪表板截图，展示了带有示例数据的 Browse Index 功能。](/img/MetalDash.png "Metal 仪表板界面")

## 快速入门

首先，请[创建 Metal 账户](https://app.getmetal.io/signup)。

然后，您可以轻松利用 `MetalRetriever` 类，开始为语义搜索、提示上下文等检索您的数据。此类需要一个 `Metal` 实例和一个参数字典，用于传递给 Metal API。

```python
from langchain.retrievers import MetalRetriever
from metal_sdk.metal import Metal


metal = Metal("API_KEY", "CLIENT_ID", "INDEX_ID");
retriever = MetalRetriever(metal, params={"limit": 2})

docs = retriever.invoke("search term")
