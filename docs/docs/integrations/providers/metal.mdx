# Metal

本页介绍如何在 LangChain 中使用 [Metal](https://getmetal.io)。

## Metal 是什么？

Metal 是一个为生产环境构建的托管检索与内存平台。您可以轻松地将数据索引到 `Metal` 中，并在其上运行语义搜索和检索。

![Metal 仪表盘截图，展示了 Browse Index 功能和示例数据。](/img/MetalDash.png "Metal 仪表盘界面")

## 快速入门

首先，请[创建您的 Metal 账户](https://app.getmetal.io/signup)。

然后，您可以轻松地利用 `MetalRetriever` 类开始为语义搜索、提示上下文等检索您的数据。此类接收一个 `Metal` 实例和一个要传递给 Metal API 的参数字典。

```python
from langchain.retrievers import MetalRetriever
from metal_sdk.metal import Metal


metal = Metal("API_KEY", "CLIENT_ID", "INDEX_ID");
retriever = MetalRetriever(metal, params={"limit": 2})

docs = retriever.invoke("search term")
```
