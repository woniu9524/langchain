# Hacker News

>[Hacker News](https://en.wikipedia.org/wiki/Hacker_News) (有时缩写为 `HN`) 是一个专注于计算机科学和创业的社交新闻网站。它由投资公司和创业孵化器 `Y Combinator` 运营。一般来说，可以提交的内容被定义为“任何能满足一个人智力好奇心的东西”。

## 安装和设置

没有特别的设置要求。
