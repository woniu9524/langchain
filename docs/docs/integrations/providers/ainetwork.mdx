# AINetwork

>[AI Network](https://www.ainetwork.ai/build-on-ain) 是一个为容纳大规模 AI 模型而设计的第 1 层区块链，它利用由 [$AIN 代币](https://www.ainetwork.ai/token) 提供支持的去中心化 GPU 网络，并丰富了由 AI 驱动的 `NFTs` (`AINFTs`)。


## 安装和设置

您需要安装 `ain-py` Python 包。

```bash
pip install ain-py
```
您需要将 `AIN_BLOCKCHAIN_ACCOUNT_PRIVATE_KEY` 环境变量设置为您的 AIN Blockchain 账户私钥。
## 工具包

请参阅[使用示例](/docs/integrations/tools/ainetwork)。

```python
from langchain_community.agent_toolkits.ainetwork.toolkit import AINetworkToolkit
```
