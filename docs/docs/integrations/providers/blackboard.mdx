# Blackboard

>[Blackboard Learn](https://en.wikipedia.org/wiki/Blackboard_Learn) (此前称为 `Blackboard Learning Management System`)
>是由 Blackboard Inc. 开发的一款基于 Web 的虚拟学习环境和学习管理系统。
>该软件具有课程管理、可定制的开放架构和可扩展的设计，允许与学生信息系统和身份验证协议集成。它可以安装在本地服务器上，由 `Blackboard ASP Solutions` 托管，或作为托管在 Amazon Web Services 上的软件即服务 (SaaS) 提供。
>其主要目的包括为传统面授课程添加在线元素，以及开发几乎不要求面授的 সম্পূর্ণ 在线课程。

## 安装和设置

没有特殊的设置要求。
