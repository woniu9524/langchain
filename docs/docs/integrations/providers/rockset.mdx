# Rockset

>[Rockset](https://rockset.com/product/) 是一个实时分析数据库服务，用于大规模提供低延迟、高并发的分析查询。它在结构化和半结构化数据上构建 Converged Index™，并为向量嵌入提供高效的存储。它支持在无模式数据上运行 SQL，使其成为运行带元数据过滤的向量搜索的理想选择。

## 安装与设置

请确保您拥有 Rockset 账户，并前往 Web 控制台获取 API 密钥。详细信息请参阅[官方网站](https://rockset.com/docs/rest-api/)。

```bash
pip install rockset
```
