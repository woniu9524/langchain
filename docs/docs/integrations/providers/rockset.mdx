# Rockset

>[Rockset](https://rockset.com/product/) 是一个实时分析数据库服务，用于大规模提供低延迟、高并发的分析查询。它在结构化和半结构化数据上构建 Converged Index™，并提供高效的向量嵌入存储。它支持在无模式数据上运行 SQL，使其成为使用元数据过滤器运行向量搜索的理想选择。

## 安装和设置

确保您拥有 Rockset 账户，并前往 Web 控制台获取 API 密钥。详细信息可在[官方网站](https://rockset.com/docs/rest-api/)上找到。

```bash
pip install rockset
```
