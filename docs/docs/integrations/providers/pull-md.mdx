# PullMd 加载器

>[PullMd](https://pull.md/) 是一项将网页转换为 Markdown 格式的服务。`langchain-pull-md` 包利用此服务将 URL（尤其是那些使用 React、Angular 或 Vue.js 等 JavaScript 框架渲染的 URL）转换为 Markdown，而无需本地渲染。

## 安装和设置

要开始使用 `langchain-pull-md`，您需要通过 pip 安装该包：

```bash
pip install langchain-pull-md
```

有关详细的集成和使用说明，请参阅[用法示例](/docs/integrations/document_loaders/pull_md)。

## 文档加载器

`langchain-pull-md` 中的 `PullMdLoader` 类提供了一种将 URL 转换为 Markdown 的便捷方法。它特别适用于加载现代 Web 应用程序的内容，以便在 LangChain 的处理能力中使用。

```python
from langchain_pull_md import PullMdLoader

# 使用 JavaScript 渲染的网页 URL 初始化加载器
loader = PullMdLoader(url='https://example.com')

# 将内容加载为 Document
documents = loader.load()

# 访问 Markdown 内容
for document in documents:
    print(document.page_content)
```

此加载器支持任何 URL，并且特别擅长处理使用动态 JavaScript 构建的网站，使其成为数据处理工作流中 Markdown 提取的通用工具。
