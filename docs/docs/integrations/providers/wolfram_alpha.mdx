# Wolfram Alpha

>[WolframAlpha](https://en.wikipedia.org/wiki/WolframAlpha) 是由 `Wolfram Research` 开发的答案引擎。
> 它通过计算外部来源的数据来回答事实性查询。

本页面介绍如何在 LangChain 中使用 `Wolfram Alpha API`。

## 安装和设置
- 使用以下命令安装所需依赖：
```bash
pip install wolframalpha
```
- 前往 Wolfram Alpha [在此处](https://developer.wolframalpha.com/) 注册开发者账户
- 创建一个应用并获取您的 `APP ID`
- 将您的 APP ID 设置为环境变量 `WOLFRAM_ALPHA_APPID`

## 封装器

### 工具类

存在一个名为 `WolframAlphaAPIWrapper` 的工具类，它封装了这个 API。要导入此工具类：

```python
from langchain_community.utilities.wolfram_alpha import WolframAlphaAPIWrapper
```

有关此工具类的更详细的演示，请参阅 [此笔记本](/docs/integrations/tools/wolfram_alpha)。

### 工具

您也可以轻松地将此工具类加载为工具（以与 Agent 一起使用）。
您可以通过以下方式实现：
```python
from langchain_community.agent_toolkits.load_tools import load_tools
tools = load_tools(["wolfram-alpha"])
```

有关工具的更多信息，请参阅 [此页面](/docs/how_to/tools_builtin)。
