# Nomic

>[Nomic](https://www.nomic.ai/) 构建的工具使每个人都能与 AI 规模的数据集进行交互，并在消费级计算机上运行 AI 模型。
>
>`Nomic` 目前提供两种产品：
>
>- `Atlas`：可视化数据引擎
>- `GPT4All`：开源边缘语言模型生态系统

Nomic 集成存在于两个合作伙伴包中：[langchain-nomic](https://pypi.org/project/langchain-nomic/)
和 [langchain-community](https://pypi.org/project/langchain-community/)。

## 安装

您可以使用以下命令安装它们：

```bash
pip install -U langchain-nomic
pip install -U langchain-community
```

## LLMs

### GPT4All

请参阅[用法示例](/docs/integrations/llms/gpt4all)。

```python
from langchain_community.llms import GPT4All
```

## Embedding models

### NomicEmbeddings

请参阅[用法示例](/docs/integrations/text_embedding/nomic)。


请参阅[用法示例](/docs/integrations/text_embedding/gpt4all)。
