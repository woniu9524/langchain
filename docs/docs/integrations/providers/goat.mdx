# GOAT

[GOAT](https://github.com/goat-sdk/goat) 是面向 AI Agent 的金融工具包。

创建能够执行以下操作的 Agent：

- 发送和接收付款
- 购买实体和数字商品及服务
- 参与各种投资策略：
  - 获取收益
  - 在预测市场上下注
- 购买加密资产
- 代币化任何资产
- 获取财务洞察

### 工作原理
GOAT 利用区块链、加密货币（如稳定币）和钱包作为基础设施，使 Agent 能够成为经济行为体：

1. 为您的 Agent 提供一个 [钱包](https://github.com/goat-sdk/goat/tree/main#chains-and-wallets)
2. 允许它在 [任何地方](https://github.com/goat-sdk/goat/tree/main#chains-and-wallets) 进行交易
3. 使用超过 [+200 种工具](https://github.com/goat-sdk/goat/tree/main#tools)

在此处 [查看 GOAT 支持的所有内容](https://github.com/goat-sdk/goat/tree/main#chains-and-wallets)。

**轻量且可扩展**
与其他工具包不同，GOAT 的设计宗旨是轻量且可扩展，其核心保持最小化，并允许您仅安装所需的工具。

如果在我们提供的超过 200 个集成中找不到所需内容，您可以轻松地：

- 创建自己的插件
- 集成新的链
- 集成新的钱包
- 集成新的 Agent 框架

在此处 [查看如何操作](https://github.com/goat-sdk/goat/tree/main#-contributing)。
