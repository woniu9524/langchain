# AssemblyAI

>[AssemblyAI](https://www.assemblyai.com/) 构建用于语音转文本、说话人分离、语音摘要等任务的 `Speech AI` 模型。
> `AssemblyAI’s` Speech AI模型包括针对语音数据（如通话、虚拟会议和播客）的准确语音转文本、说话人检测、情感分析、章节检测、PII  redaction。


## 安装和设置

获取你的 [API 密钥](https://www.assemblyai.com/dashboard/signup)。

安装 `assemblyai` 包。

```bash
pip install -U assemblyai
```

## 文档加载器

### AssemblyAI 音频转录

`AssemblyAIAudioTranscriptLoader` 使用 `AssemblyAI API` 转录音频文件，并将转录的文本加载到文档中。

查看 [用法示例](/docs/integrations/document_loaders/assemblyai)。

```python
from langchain_community.document_loaders import AssemblyAIAudioTranscriptLoader
```

### AssemblyAI 按 ID 加载音频

`AssemblyAIAudioLoaderById` 使用 AssemblyAI API 获取现有转录内容，并将转录的文本根据指定的格式加载到一个或多个文档中。

```python
from langchain_community.document_loaders import AssemblyAIAudioLoaderById
```
