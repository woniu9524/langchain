# Fauna

>[Fauna](https://fauna.com/) 是一个分布式文档关系型数据库，它结合了文档的灵活性和关系型数据库的强大功能，该数据库符合 ACID 标准，并可在区域、云或全球范围内进行扩展。

## 安装和设置

我们需要获取密钥。
请参阅详细的[指南](https://docs.fauna.com/fauna/current/learn/security_model/)。

我们需要安装 `fauna` 包。

```bash
pip install -U fauna
```
