# Fauna

>[Fauna](https://fauna.com/) 是一个分布式文档关系数据库，它将文档的灵活性与关系型、ACID 兼容且跨区域、跨云或全球扩展的数据库的强大功能相结合。
 

## 安装与设置

我们需要获取密钥。
请参阅详细的 [指南](https://docs.fauna.com/fauna/current/learn/security_model/)。

我们需要安装 `fauna` 包。

```bash
pip install -U fauna
```

## 文档加载器

请参阅 [用法示例](/docs/integrations/document_loaders/fauna)。

from langchain_community.document_loaders.fauna import FaunaLoader
