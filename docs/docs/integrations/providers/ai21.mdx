# AI21 Labs

>[AI21 Labs](https://www.ai21.com/about) 是一家专注于自然语言处理 (NLP) 的公司，该公司开发能够理解和生成自然语言的 AI 系统。

本页面将介绍如何在 `LangChain` 中使用 `AI21` 生态系统。

## 安装和设置

- 获取 AI21 API 密钥并将其设置为环境变量 (`AI21_API_KEY`)
- 安装 Python 包：

```bash
pip install langchain-ai21
```

## 聊天模型

### AI21 Chat 

请参阅[用法示例](/docs/integrations/chat/ai21)。

```python
from langchain_ai21 import ChatAI21
```
