# Apify


>[Apify](https://apify.com) 是一个用于网络抓取和数据提取的云平台，
>它提供了一个包含一千多个现成应用程序的[生态系统](https://apify.com/store)，这些应用程序称为*Actors*，可用于各种抓取、爬取和提取用例。

[![Apify Actors](/img/ApifyActors.png)](https://apify.com/store)

此集成使您可以在 `Apify` 平台上运行 Actors，并将其结果加载到 LangChain 中，以便使用来自网络（例如，从具有文档、博客或知识库的网站生成答案）的文档和数据来填充您的向量索引。


## 安装和设置

- 使用以下命令安装 LangChain Apify Python 包：
```bash
pip install langchain-apify
```
- 获取您的 [Apify API 令牌](https://console.apify.com/account/integrations)，然后将其设置为环境变量 (`APIFY_API_TOKEN`)，或在构造函数中将其作为 `apify_api_token` 传递。

## 工具

您可以使用 `ApifyActorsTool` 将 Apify Actors 与代理一起使用。

```python
from langchain_apify import ApifyActorsTool
```

有关示例用法，请参阅[此笔记本](/docs/integrations/tools/apify_actors)，以及在 [Apify LangGraph 代理 Actor 模板](https://apify.com/templates/python-langgraph) 中使用 LangGraph 的完整工具调用代理示例。

有关如何使用此工具的更多信息，请访问[Apify 集成文档](https://docs.apify.com/platform/integrations/langgraph)。

## Wrapper

您可以使用 `ApifyWrapper` 在 Apify 平台上运行 Actors。

```python
from langchain_apify import ApifyWrapper
```

有关如何使用此包装器的更多信息，请参阅[Apify 集成文档](https://docs.apify.com/platform/integrations/langchain)。


## 文档加载器

您还可以使用我们的 `ApifyDatasetLoader` 从 Apify 数据集中获取数据。

```python
from langchain_apify import ApifyDatasetLoader
```

有关此加载器的更详细演练，请参阅[此笔记本](/docs/integrations/document_loaders/apify_dataset)。


此集成的源代码可以在 [LangChain Apify 存储库](https://github.com/apify/langchain-apify) 中找到。
