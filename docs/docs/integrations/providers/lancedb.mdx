# LanceDB

此页面涵盖了如何在 LangChain 中使用 [LanceDB](https://github.com/lancedb/lancedb)。
它分为两部分：安装与设置，然后是特定 LanceDB 包装器的参考。

## 安装与设置

- 使用 `pip install lancedb` 安装 Python SDK

## 包装器

### VectorStore

存在一个围绕 LanceDB 数据库的包装器，允许您将其用作向量存储，无论是用于语义搜索还是示例选择。

导入此向量存储：

```python
from langchain_community.vectorstores import LanceDB
```

有关 LanceDB 包装器的更详细教程，请参阅[此笔记本](/docs/integrations/vectorstores/lancedb)
