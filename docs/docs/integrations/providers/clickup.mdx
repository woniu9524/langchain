# ClickUp

>[ClickUp](https://clickup.com/) 是一个集成型效率平台，为各行各业的大小团队提供灵活且可定制的工作管理解决方案、工具和功能。
>
>它是一款面向各类规模企业的云端项目管理解决方案，具备沟通和协作工具，有助于实现组织目标。

## 安装和设置

1. 创建一个 [ClickUp App](https://help.clickup.com/hc/en-us/articles/6303422883095-Create-your-own-app-with-the-ClickUp-API)
2. 遵循 [这些步骤](https://clickup.com/api/developer-portal/authentication/) 获取你的 client_id 和 client_secret。
