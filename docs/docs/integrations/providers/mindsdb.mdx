# MindsDB

MindsDB 是一个用于定制企业数据的 AI 平台。借助 MindsDB 及其近 200 个与[数据源](https://docs.mindsdb.com/integrations/data-overview)和[AI/ML 框架](https://docs.mindsdb.com/integrations/ai-overview)的集成，任何开发人员都可以利用其企业数据，更快、更安全地为自身目的定制 AI。

借助 MindsDB，您可以将任何数据源连接到任何 AI/ML 模型，以实现和自动化 AI 驱动的应用程序。利用来自数据库、向量存储或应用程序的数据，实时部署、服务和微调模型。所有这些都可以使用开发人员已知的通用工具来完成。

MindsDB 与 LangChain 集成，使用户能够：

- 在 MindsDB 中部署 LangChain 可用的模型，使其能够访问众多数据源。
- 在 MindsDB 中使用 LangChain 可用的模型，并利用实时动态数据进行微调。
- 使用 LangChain 和 MindsDB 自动化 AI 工作流。

请关注[我们的文档](https://docs.mindsdb.com/integrations/ai-engines/langchain)，了解有关 MindsDB 与 LangChain 集成的更多信息并查看示例。
