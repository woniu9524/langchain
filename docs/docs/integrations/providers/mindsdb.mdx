# MindsDB

MindsDB 是一个用于定制企业数据的 AI 平台。借助 MindsDB 及其近 200 个与 [数据源](https://docs.mindsdb.com/integrations/data-overview) 和 [AI/ML 框架](https://docs.mindsdb.com/integrations/ai-overview) 的集成，任何开发者都可以使用其企业数据，更快、更安全地定制满足其需求的 AI。

通过 MindsDB，您可以将任何数据源连接到任何 AI/ML 模型，以实现和自动化 AI 驱动的应用程序。利用来自数据库、向量存储或应用程序的数据，实时部署、服务和微调模型。所有这些都可以使用开发人员已知的通用工具来完成。

MindsDB 与 LangChain 集成，使用户可以：

- 在 MindsDB 中部署 LangChain 可用的模型，使其能够访问众多数据源。
- 在 MindsDB 中使用实时和动态数据微调 LangChain 可用的模型。
- 使用 LangChain 和 MindsDB 自动化 AI 工作流程。

请遵循 [我们的文档](https://docs.mindsdb.com/integrations/ai-engines/langchain) 以了解更多关于 MindsDB 与 LangChain 集成的知识，并查看示例。
