# ElevenLabs

>[ElevenLabs](https://elevenlabs.io/about) 是一家语音人工智能研究与部署公司，
> 其使命是让内容能够以任何语言和声音被普遍访问。
>
> `ElevenLabs` 可生成最逼真、用途最广泛且能理解上下文的 AI 音频，
> 能够在 29 种语言中以数百种新旧声音生成语音。

## 安装和设置

首先，您需要设置一个 ElevenLabs 账户。您可以遵循
[此处的说明](https://docs.elevenlabs.io/welcome/introduction)。

安装 Python 包：

```bash
pip install elevenlabs
```

## 工具

请参阅[使用示例](/docs/integrations/tools/eleven_labs_tts)。
