# ElevenLabs

>[ElevenLabs](https://elevenlabs.io/about) 是一家语音 AI 研究与部署公司，
>其使命是让内容能够以任何语言和声音被普遍访问。
>
>`ElevenLabs` 创造出最逼真、最多功能且具备上下文感知能力的
>AI 音频，能够以数百种现有和新增的声音生成 29 种语言的语音。

## 安装与设置

首先，您需要设置一个 ElevenLabs 账户。您可以遵循
[此处的说明](https://docs.elevenlabs.io/welcome/introduction)。

安装 Python 包：

```bash
pip install elevenlabs
```
