# E2B

>[E2B](https://e2b.dev/) 提供开源的安全沙箱，用于 AI 生成的代码执行。在此了解更多信息：[https://github.com/e2b-dev](https://github.com/e2b-dev)。

## 安装和设置

您需要安装一个 Python 包：

```bash
pip install e2b_code_interpreter
```

## 工具

请参阅[用法示例](/docs/integrations/tools/e2b_data_analysis)。

```python
from langchain_community.tools import E2BDataAnalysisTool
```
