# E2B

>[E2B](https://e2b.dev/) 提供开源的安全沙箱，用于 AI 生成的代码执行。更多信息请参见 [此处](https://github.com/e2b-dev)。

## 安装与设置

您需要安装一个 Python 包：

```bash
pip install e2b_code_interpreter
```

## 工具

请参见 [用法示例](/docs/integrations/tools/e2b_data_analysis)。

```python
from langchain_community.tools import E2BDataAnalysisTool
