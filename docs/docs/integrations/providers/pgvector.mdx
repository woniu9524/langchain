# PGVector

本页介绍如何在 LangChain 中使用 Postgres [PGVector](https://github.com/pgvector/pgvector) 生态系统。
它分为两个部分：安装和设置，然后是特定 PGVector 包装器的参考。

## 安装
- 使用 `pip install pgvector` 安装 Python 包

## 设置
1. 第一步是创建已安装 `pgvector` 扩展的数据库。

    请遵循 [PGVector 安装步骤](https://github.com/pgvector/pgvector#installation) 中的步骤来安装数据库和扩展。Docker 镜像是在开始时最简单的方法。

## 包装器

### VectorStore

有一个围绕 Postgres 向量数据库的包装器，允许您将其用作向量存储，无论是用于语义搜索还是示例选择。

导入此向量存储：
```python
from langchain_community.vectorstores.pgvector import PGVector
```
