# PGVector

本页介绍了如何在 LangChain 中使用 Postgres [PGVector](https://github.com/pgvector/pgvector) 生态系统。
内容分为两部分：安装与设置，以及对特定 PGVector 包装器的引用。

## 安装
- 使用 `pip install pgvector` 安装 Python 包。


## 设置
1. 第一步是创建一个安装了 `pgvector` 扩展的数据库。

    请按照 [PGVector 安装步骤](https://github.com/pgvector/pgvector#installation) 中的说明来安装数据库和扩展。使用 Docker 镜像是最简单的入门方式。

## 包装器

### 向量存储

我们提供了一个围绕 Postgres 向量数据库的包装器，让您可以将其用作向量存储，以进行语义搜索或示例选择。

要导入此向量存储：
```python
from langchain_community.vectorstores.pgvector import PGVector
```
