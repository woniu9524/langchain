# GeoPandas

>[GeoPandas](https://geopandas.org/) 是一个开源项目，旨在简化在 Python 中使用地理空间数据的过程。`GeoPandas` 扩展了 `pandas` 使用的数据类型，以便对几何类型执行空间操作。几何操作由 `shapely` 执行。

## 安装和设置

我们需要安装几个 Python 包。

```bash
pip install -U sodapy pandas geopandas
```
