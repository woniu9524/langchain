# SparkLLM

>[SparkLLM](https://xinghuo.xfyun.cn/spark) 是科大讯飞独立研发的大规模认知模型。
它通过学习大量的文本、代码和图像，具备跨领域知识和语言理解能力。
它可以根据自然对话理解和执行任务。

## Chat models

请参阅 [使用示例](/docs/integrations/chat/sparkllm)。

```python
from langchain_community.chat_models import ChatSparkLLM
```

## LLMs

请参阅 [使用示例](/docs/integrations/llms/sparkllm)。

```python
from langchain_community.llms import SparkLLM
```

## Embedding models

请参阅 [使用示例](/docs/integrations/text_embedding/sparkllm)

from langchain_community.embeddings import SparkLLMTextEmbeddings
