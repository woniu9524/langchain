# GooseAI

>[GooseAI](https://goose.ai) 让部署 NLP 服务更轻松、更易于访问。
> `GooseAI` 是一个通过 API 交付的完全托管的推理服务。
> `GooseAI` 具有与其他知名 API 的功能对等性，通过简单地更改代码中的两行内容，为服务开源语言模型提供了即插即用的解决方案，并实现了业界最优的经济效益。

## 安装和设置

- 使用 `pip install openai` 安装 Python SDK
- 从此链接 [here](https://goose.ai/) 获取您的 GooseAI api 密钥。
- 设置环境变量 (`GOOSEAI_API_KEY`)。

```python
import os
os.environ["GOOSEAI_API_KEY"] = "YOUR_API_KEY"
```
