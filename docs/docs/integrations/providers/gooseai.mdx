# GooseAI

>[GooseAI](https://goose.ai) 使部署 NLP 服务更加轻松便捷。
> `GooseAI` 是一个通过 API 交付的全托管推理服务。
> 凭借与其他知名 API 的功能对等性，`GooseAI` 提供了一个即插即用的解决方案，
> 只需更改代码中的两行，即可以行业最佳的经济效益来提供开源语言模型服务。

## 安装和设置

- 使用 `pip install openai` 安装 Python SDK
- 从此链接 [此处](https://goose.ai/) 获取您的 GooseAI API 密钥。
- 设置环境变量 (`GOOSEAI_API_KEY`)。

```python
import os
os.environ["GOOSEAI_API_KEY"] = "YOUR_API_KEY"
```
