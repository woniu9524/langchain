# Argilla

>[Argilla](https://argilla.io/) 是一个开源的 LLM 数据策展平台。
> 使用 `Argilla`，每个人都可以通过更快速的数据策展来构建强大的语言模型，
> 该平台支持 LLM 的整个 MLOps 周期，从数据标注到模型监控。

## 安装和设置

获取您的 [API 密钥](https://platform.openai.com/account/api-keys)。

安装 Python 包：

```bash
pip install argilla
```

## 回调

```python
from langchain.callbacks import ArgillaCallbackHandler
```

请参阅 [示例](/docs/integrations/callbacks/argilla)。
