# Typesense

> [Typesense](https://typesense.org) 是一个开源的内存搜索引擎，您可以选择 [自行托管](https://typesense.org/docs/guide/install-typesense.html#option-2-local-machine-self-hosting) 或在 [Typesense Cloud](https://cloud.typesense.org/) 上运行。
> `Typesense` 通过将整个索引存储在 RAM 中（并有磁盘备份）来专注于性能，同时还通过简化可用选项和设置良好默认值来 fokus 开发者体验。

## 安装和设置

```bash
pip install typesense openapi-schema-pydantic
```
