# BagelDB

> [BagelDB](https://www.bageldb.ai/)（`面向 AI 的开放式向量数据库`），就像是 AI 数据领域的 GitHub。
它是一个协作平台，用户可以在其中创建、
共享和管理向量数据集。它既可以支持独立开发者的私有项目，
也可以支持企业的内部协作，还能支持数据 DAO 的公开贡献。

## 安装与设置

```bash
pip install betabageldb
```
