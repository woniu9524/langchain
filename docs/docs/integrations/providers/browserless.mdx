# Browserless

>[Browserless](https://www.browserless.io/docs/start) 是一项允许你在云端运行无头 Chrome 实例的服务。这是大规模运行基于浏览器的自动化操作的绝佳方式，无需担心管理自己的基础设施。

## 安装和设置

我们需要在此处 [获取 API 密钥](https://www.browserless.io/pricing/)。
