# Browserless

>[Browserless](https://www.browserless.io/docs/start) 是一个允许您在云端运行无头 Chrome 实例的服务。它是大规模运行基于浏览器的自动化而无需担心管理自身基础设施的绝佳方式。

## 安装和设置

我们必须在此处获取 API 密钥：[https://www.browserless.io/pricing/](https://www.browserless.io/pricing/)。
