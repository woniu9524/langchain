# Tair

>[Alibaba Cloud Tair](https://www.alibabacloud.com/help/en/tair/latest/what-is-tair) 是阿里云开发的一款云原生内存数据库服务，
>它提供了丰富的数据模型和企业级功能，以支持您的实时在线场景，同时保持与开源 `Redis` 的完全兼容。
>`Tair` 还引入了基于新型非易失性存储介质 (NVM) 的持久化内存优化实例。

## 安装和设置

安装 Tair Python SDK：

```bash
pip install tair
```

## 向量存储

```python
from langchain_community.vectorstores import Tair
```

请参阅 [使用示例](/docs/integrations/vectorstores/tair)。
