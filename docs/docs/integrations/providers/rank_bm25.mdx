# rank_bm25

[rank_bm25](https://github.com/dorianbrown/rank_bm25) 是一个开源算法集合，
用于查询文档并返回最相关的文档，常用于构建搜索引擎。

有关可用算法，请参阅其[项目页面](https://github.com/dorianbrown/rank_bm25)。


## 安装和设置

首先，您需要安装 `rank_bm25` python 包。

```bash
pip install rank_bm25
```


## Retriever

请参阅[使用示例](/docs/integrations/retrievers/bm25)。

```python
from langchain_community.retrievers import BM25Retriever
```
