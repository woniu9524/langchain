# SemaDB

>[SemaDB](https://semafind.com/) 是一个简单易用的向量相似性搜索引擎。它提供低成本的云托管版本，帮助您轻松构建 AI 应用程序。

通过我们的托管版本 SemaDB Cloud，简单易用意味着无需进行 Pod 规模计算、模式定义、分区设置、参数调优、搜索算法调优、复杂安装或复杂的 API。它已集成 [RapidAPI](https://rapidapi.com/semafind-semadb/api/semadb)，提供透明计费、自动分片和交互式 API 调试工具。

## 安装

无需安装，直接在 [RapidAPI](https://rapidapi.com/semafind-semadb/api/semadb) 上开始使用 SemaDB Cloud。

## Vector Store

有一个基础的 Wrapper 围绕着 `SemaDB` collections，允许您将其用作 vectorstore。

```python
from langchain_community.vectorstores import SemaDB
```

您可以在[此笔记本](/docs/integrations/vectorstores/semadb)中找到有关如何使用该 Wrapper 的教程。
