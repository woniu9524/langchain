# SemaDB

>[SemaDB](https://semafind.com/) 是一个简单易用的向量相似性搜索引擎。它提供低成本的云托管版本，帮助您轻松构建 AI 应用程序。

通过我们的托管版本 SemaDB Cloud，简单易用意味着无需进行 Pod 大小计算、无需定义 Schema、无需配置分区、无需调整参数、无需调整搜索算法、无需复杂安装、无需复杂 API。它已集成 [RapidAPI](https://rapidapi.com/semafind-semadb/api/semadb)，提供透明的计费、自动分片和交互式 API 调试工具。

## 安装

无需安装，直接在 [RapidAPI](https://rapidapi.com/semafind-semadb/api/semadb) 上开始使用 SemaDB Cloud。

## Vector Store

我们提供了一个围绕 `SemaDB` 集合的基本封装，允许您将其用作向量存储。

```python
from langchain_community.vectorstores import SemaDB
```

您可以在[此笔记本](/docs/integrations/vectorstores/semadb)中找到有关如何使用此封装的教程。
