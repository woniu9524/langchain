# Apple

> Apple Inc. (Wikipedia) 是一家美国跨国公司和科技公司。
>
> iMessage (Wikipedia) 是由 Apple Inc. 开发并于 2011 年推出的即时通讯服务。
> `iMessage` 仅在 Apple 平台上运行。

## 安装和设置

请参阅[设置说明](/docs/integrations/chat_loaders/imessage)。

## Chat loader

它从 `iMessage` 的 `chat.db` `SQLite` 文件加载聊天会话。

请参阅[用法示例](/docs/integrations/chat_loaders/imessage)。

```python
from langchain_community.chat_loaders.imessage import IMessageChatLoader
```
