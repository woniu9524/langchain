# Twitter

>[Twitter](https://twitter.com/) 是一项在线社交媒体和社交网络服务。


## 安装和设置

```bash
pip install tweepy
```

我们必须使用 `Twitter API` token 初始化加载器，并且需要设置 Twitter 的 `username`。


## 文档加载器

请参阅 [使用示例](/docs/integrations/document_loaders/twitter)。

```python
from langchain_community.document_loaders import TwitterTweetLoader
```
