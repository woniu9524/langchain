# Spreedly

>[Spreedly](https://docs.spreedly.com/) 是一项服务，可让您安全地存储信用卡，并利用这些信用卡与任意数量的支付网关和第三方 API 进行交易。它通过同时提供信用卡标记化/存储服务以及网关和接收方集成服务来实现这一点。由 Spreedly 进行标记化的支付方式存储在 `Spreedly` 中，让您可以独立存储信用卡，然后根据业务需求将该信用卡传递给不同的终端。
