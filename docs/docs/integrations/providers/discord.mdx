# Discord (社区加载器)

>[Discord](https://discord.com/) 是一个VoIP和即时通讯社交平台。用户可以通过私聊或称为“服务器”的社区进行语音通话、视频通话、文本消息、媒体和文件交流。服务器是持久聊天室和语音频道的集合，可以通过邀请链接访问。

## 安装和设置

```bash
pip install pandas
```

请按照以下步骤下载您的 `Discord` 数据：

1. 前往您的 **用户设置**
2. 然后前往 **隐私与安全**
3. 找到 **请求我的所有数据** 并点击 **请求数据** 按钮

您可能需要 30 天才能收到数据。您会收到一封发送到您 Discord 注册邮箱的邮件。该邮件将包含一个下载按钮，您可以使用该按钮下载您的个人 Discord 数据。
