# Discord (社区加载器)

>[Discord](https://discord.com/) 是一个VoIP和即时通讯的社交平台。用户可以通过语音通话、视频通话、短信、媒体和文件在私人聊天中或作为“服务器”社区的一部分进行交流。服务器是可以通过邀请链接访问的持久聊天室和语音频道的集合。

## 安装和设置

```bash
pip install pandas
```

按照以下步骤下载您的 `Discord` 数据：

1. 转到您的 **用户设置**
2. 然后转到 **隐私与安全**
3. 前往 **请求我所有的数据** 并点击 **请求数据** 按钮

您可能需要 30 天才能收到您的数据。您将收到一封电子邮件，发送到您在 Discord 注册的地址。该电子邮件将包含一个下载按钮，您可以使用它下载您的个人 Discord 数据。
