# FMP Data (金融数据准备)

> [FMP-Data](https://pypi.org/project/fmp-data/) 是一个用于连接 Financial Data Prep API 的 Python 包。它简化了如何访问生产级别数据的过程。


## 安装和设置

通过访问[此页面](https://site.financialmodelingprep.com/pricing-plans?couponCode=mehdi)获取 `FMP Data` API 密钥，并将其设置为环境变量 (`FMP_API_KEY`)。

然后，安装[langchain-fmp-data](https://pypi.org/project/langchain-fmp-data/)。

## 工具

请参阅[示例](https://github.com/MehdiZare/langchain-fmp-data/tree/main/docs)。

```python
from langchain_fmp_data import FMPDataTool, FMPDataToolkit
```
