# Stack Exchange

>[Stack Exchange](https://en.wikipedia.org/wiki/Stack_Exchange) 是一个问答 (Q&A) 网站网络，涵盖各种领域的特定主题，每个网站都涵盖一个特定主题，用户可以通过声誉奖励过程获得问题、答案和用户。

本页介绍如何在 LangChain 中使用 `Stack Exchange API`。

## 安装和设置
- 使用以下命令安装要求：
```bash
pip install stackapi
```

## 封装器

### 工具

存在一个 StackExchangeAPIWrapper 工具封装器来封装此 API。导入此工具：

```python
from langchain_community.utilities import StackExchangeAPIWrapper
```

有关此封装器的更详细的演练，请参阅[此笔记本](/docs/integrations/tools/stackexchange)。

### 工具

您也可以轻松地将此封装器加载为工具（与 Agent 一起使用）。
您可以通过以下方式完成此操作：
```python
from langchain_community.agent_toolkits.load_tools import load_tools
tools = load_tools(["stackexchange"])
```

有关工具的更多信息，请参阅[此页面](/docs/how_to/tools_builtin)。
