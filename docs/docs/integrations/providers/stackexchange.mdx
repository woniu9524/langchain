# Stack Exchange

>[Stack Exchange](https://en.wikipedia.org/wiki/Stack_Exchange) 是一个问答 (Q&A) 网站网络，涵盖了众多领域的知识，每个网站都侧重于一个特定主题，其问题、答案以及用户都遵循一套声誉奖励机制。

本页面将介绍如何在 LangChain 中使用 `Stack Exchange API`。

## 安装和设置
- 使用以下命令安装依赖：
```bash
pip install stackapi
```

## 封装器（Wrappers）

### 实用工具（Utility）

存在一个名为 `StackExchangeAPIWrapper` 的实用工具，它封装了此 API。导入此实用工具的方法如下：

```python
from langchain_community.utilities import StackExchangeAPIWrapper
```

有关此封装器的更详细教程，请参阅[此 Notebook](/docs/integrations/tools/stackexchange)。

### 工具（Tool）

您也可以轻松地将此封装器加载为工具（以与 Agent 配合使用）。
您可以这样做：
```python
from langchain_community.agent_tool_kits.load_tools import load_tools
tools = load_tools(["stackexchange"])
```

有关工具的更多信息，请参阅[此页面](/docs/how_to/tools_builtin)。
