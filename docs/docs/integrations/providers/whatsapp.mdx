# WhatsApp

>[WhatsApp](https://www.whatsapp.com/)（也称为 `WhatsApp Messenger`）是一款免费的、跨平台的、集中的即时消息（IM）和网络语音通话（VoIP）服务。它允许用户发送文本和语音消息、进行语音和视频通话，以及共享图片、文档、用户位置和其他内容。


## 安装和设置

无需进行任何特殊设置。
