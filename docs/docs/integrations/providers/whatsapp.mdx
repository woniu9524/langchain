# WhatsApp

>[WhatsApp](https://www.whatsapp.com/)（也称为 `WhatsApp Messenger`）是一款免费、跨平台、集中式的即时消息 (IM) 和网络语音服务 (VoIP)。它允许用户发送文本和语音消息、进行语音和视频通话，以及共享图片、文档、用户位置和其他内容。


## 安装和设置

没有特殊的设置要求。
