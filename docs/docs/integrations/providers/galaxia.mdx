# Smabbler
> Smabbler 的图驱动平台通过将数据转化为结构化的知识基础来加速 AI 开发。

# Galaxia

> Galaxia 知识库是用于检索增强生成 (RAG) 的集成知识库和检索机制。与标准解决方案不同，它基于使用符号自然语言处理和知识表示解决方案构建的知识图。提供的文本会被分析并转化为包含文本、语言和语义信息的图。这种丰富的结构允许基于语义信息进行检索，而不是基于向量相似性/距离。

使用 Galaxia 实现 RAG 涉及首先将文件上传到 [Galaxia](https://beta.cloud.smabbler.com/home)，在那里进行分析，然后构建模型（知识图）。模型构建完成后，您可以使用 `GalaxiaRetriever` 连接到 API 并开始检索。

更多信息：[docs](https://smabbler.gitbook.io/smabbler)

## 安装
```
pip install langchain-galaxia-retriever
```

## 用法

```
from langchain_galaxia_retriever.retriever import GalaxiaRetriever

gr = GalaxiaRetriever(
    api_url="beta.api.smabbler.com",
    api_key="<key>",
    knowledge_base_id="<knowledge_base_id>",
    n_retries=10,
    wait_time=5,
)

result = gr.invoke('<test question>')
print(result)
```
