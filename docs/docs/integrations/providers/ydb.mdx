# YDB

与 YDB 相关的所有功能。

> [YDB](https://ydb.tech/) 是一个多功能开源分布式 SQL 数据库，结合了高可用性和可扩展性与强一致性和 ACID 事务。
> 它可以同时处理事务性 (OLTP)、分析性 (OLAP) 和流式工作负载。

## 安装和设置

```bash
pip install langchain-ydb
```

## 向量存储

导入 YDB 向量存储：

```python
from langchain_ydb.vectorstores import YDB
```

有关 YDB 向量存储的更详细演练，请参阅[此笔记本](/docs/integrations/vectorstores/ydb)。
