# OpenWeatherMap

>[OpenWeatherMap](https://openweathermap.org/api/) 提供特定地点所有关键的天气数据：
>- 当前天气
>- 1小时分钟级预报
>- 48小时小时级预报
>- 8天日预报
>- 国家天气警报
>- 40多年前的历史天气数据

本页面涵盖了如何在 LangChain 中使用 `OpenWeatherMap API`。

## 安装与设置

- 使用以下命令安装所需依赖：
```bash
pip install pyowm
```
- 前往 OpenWeatherMap 注册账号并在此处获取您的 API 密钥：[OpenWeatherMap API](https://openweathermap.org/api/)
- 将您的 API 密钥设置为 `OPENWEATHERMAP_API_KEY` 环境变量

## 封装器 (Wrappers)

### 实用工具 (Utility)

存在一个名为 OpenWeatherMapAPIWrapper 的实用工具封装了此 API。要导入此工具：

```python
from langchain_community.utilities.openweathermap import OpenWeatherMapAPIWrapper
```

有关此封装器的更详细的介绍，请参阅 [此 Notebook](/docs/integrations/tools/openweathermap)。

### 工具 (Tool)

您也可以轻松地将此封装器加载为工具（用于 Agent）。
您可以通过以下方式实现：

```python
import os
from langchain_community.utilities import OpenWeatherMapAPIWrapper

os.environ["OPENWEATHERMAP_API_KEY"] = ""
weather = OpenWeatherMapAPIWrapper()
tools = [weather.run]
```

有关工具的更多信息，请参阅 [此页面](/docs/how_to/tools_builtin)。
