# OpenWeatherMap

>[OpenWeatherMap](https://openweathermap.org/api/) 提供特定地点所有必需的天气数据：
>- 当前天气
>- 1小时分钟级预报
>- 48小时小时级预报
>- 8天每日预报
>- 国家天气警报
>- 40多年份的历史天气数据

本页介绍如何在 LangChain 中使用 `OpenWeatherMap API`。

## 安装和设置

- 使用以下命令安装所需依赖：
```bash
pip install pyowm
```
- 前往 OpenWeatherMap 注册账户并在此处获取您的 API 密钥：[https://openweathermap.org/api/](https://openweathermap.org/api/)
- 将您的 API 密钥设置为 `OPENWEATHERMAP_API_KEY` 环境变量

## 封装器

### 工具类

存在一个 OpenWeatherMapAPIWrapper 工具类来封装此 API。导入此工具类：

```python
from langchain_community.utilities.openweathermap import OpenWeatherMapAPIWrapper
```

有关此封装器的更详细教程，请参阅 [此笔记本](/docs/integrations/tools/openweathermap)。

### 工具

您也可以轻松地将此封装器加载为工具（用于 Agent）。
您可以通过以下方式实现：

```python
import os
from langchain_community.utilities import OpenWeatherMapAPIWrapper

os.environ["OPENWEATHERMAP_API_KEY"] = ""
weather = OpenWeatherMapAPIWrapper()
tools = [weather.run]
```

有关工具的更多信息，请参阅 [此页面](/docs/how_to/tools_builtin)。
