# IBM

LangChain 与 IBM 技术相关的集成，包括
[IBM watsonx.ai](https://www.ibm.com/products/watsonx-ai) 平台和 DB2 数据库。

## Watsonx AI
IBM® watsonx.ai™ AI Studio 是 IBM [watsonx](https://www.ibm.com/watsonx)™ AI 和数据平台的一部分，它将由 [基础模型](https://www.ibm.com/products/watsonx-ai/foundation-models) 提供支持的新型生成式 AI 功能与传统机器学习 (ML) 相结合，构建一个跨越 AI 生命周期的强大工作室。通过易于使用的工具，使用您的企业数据调整和指导模型，以满足您的需求，从而构建和优化高性能的提示。借助 watsonx.ai，您可以以更少的时间和更少的数据构建 AI 应用程序。
Watsonx.ai 提供：

- **多模型多样性和灵活性：** 可从 IBM 开发的模型、开源模型和第三方模型中进行选择，或构建您自己的模型。
- **差异化的客户保护：** IBM 为 IBM 开发的模型提供支持，并为客户就第三方知识产权索赔提供赔偿。
- **端到端 AI 治理：** 企业可以使用跨业务的可信数据来扩展和加速 AI 的影响，无论数据驻留在何处。
- **混合、多云部署：** IBM 提供灵活性，可将您的 AI 工作负载集成并部署到您选择的混合云堆栈中。


### 安装和设置

使用以下命令安装集成包：
```bash
pip install -qU langchain-ibm
```

获取 IBM watsonx.ai API 密钥，并将其设置为环境变量 (`WATSONX_APIKEY`)
```python
import os

os.environ["WATSONX_APIKEY"] = "your IBM watsonx.ai api key"
```

## DB2

### Vector stores

#### IBM DB2 Vector Store 和 Vector Search

IBM DB2 关系数据库 v12.1.2 及更高版本提供了向量存储和向量搜索的功能。安装 `langchain-db2` 包将为 Langchain 用户提供 DB2 向量存储和向量搜索的支持。

请在此指南 [here](/docs/integrations/vectorstores/db2) 中查看详细的使用示例。

安装：这是一个仅用于向量存储功能的独立包，可以在没有 `langchain-ibm` 包的情况下运行。
```bash
pip install -U langchain-db2
```
用法：
```python
from langchain_db2 import db2vs
from langchain_db2.db2vs import DB2VS
