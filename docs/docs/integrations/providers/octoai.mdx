# OctoAI

>[OctoAI](https://docs.octoai.cloud/docs) 提供便捷高效的计算资源访问，并允许用户将自选的 AI 模型集成到应用程序中。
> `OctoAI` 计算服务可帮助您轻松运行、调优和扩展 AI 应用程序。


## 安装与设置

- 安装 `openai` Python 包：
  ```bash
  pip install openai
  ````
- 在 `OctoAI` 上注册并从[您的 OctoAI 账户页面](https://octoai.cloud/settings)获取 API Token。


## Chat models

参见[用法示例](/docs/integrations/chat/octoai)。

```python
from langchain_community.chat_models import ChatOctoAI
```
