# OpenAI

所有与 OpenAI 相关的功能

> [OpenAI](https://en.wikipedia.org/wiki/OpenAI) 是位于美国的人工智能（AI）研究实验室，
> 由非营利组织 **OpenAI Incorporated** 及其营利性子公司 **OpenAI Limited Partnership** 组成。
> **OpenAI** 的宗旨是促进和发展友好的 AI。
> **OpenAI** 的系统运行在 **Microsoft** 的 **Azure** 超级计算平台上。
>
> [OpenAI API](https://platform.openai.com/docs/models) 由一套功能和价格各异的模型提供支持。
>
> [ChatGPT](https://chat.openai.com) 是由 `OpenAI` 开发的人工智能（AI）聊天机器人。

## 安装与设置

使用以下命令安装集成包
```bash
pip install langchain-openai
```

获取 OpenAI API 密钥并将其设置为环境变量（`OPENAI_API_KEY`）

## Chat model (聊天模型)

请参阅 [使用示例](/docs/integrations/chat/openai)。

```python
from langchain_openai import ChatOpenAI
```

如果您使用的是托管在 `Azure` 上的模型，则应使用不同的包装器：
```python
from langchain_openai import AzureChatOpenAI
```

## Tools (工具)

### Dall-E Image Generator (Dall-E 图像生成器)

>[OpenAI Dall-E](https://openai.com/dall-e-3) 是 `OpenAI` 开发的文本到图像模型，
>它使用深度学习方法，根据自然语言描述（称为“提示”）生成数字图像。


请参阅 [使用示例](/docs/integrations/tools/dalle_image_generator)。

```python

```

## Tokenizer (分词器)

您可以在多个地方使用 `tiktoken` 分词器。默认情况下，它用于计算 OpenAI LLM 的 token 数量。

您还可以使用它在分割文档时计算 token 数量
```python
from langchain.text_splitter import CharacterTextSplitter
CharacterTextSplitter.from_tiktoken_encoder(...)
```
有关此功能的更详细教程，请参阅 [此笔记本](/docs/how_to/split_by_token/#tiktoken)
