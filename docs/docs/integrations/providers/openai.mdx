# OpenAI

与OpenAI相关的所有功能

> [OpenAI](https://en.wikipedia.org/wiki/OpenAI) 是美国人工智能（AI）研究实验室，
> 由非营利组织 **OpenAI Incorporated**
> 及其营利性子公司 **OpenAI Limited Partnership** 组成。
> **OpenAI** 进行AI研究，其公开声明的意图是促进和开发友好的AI。
> **OpenAI** 系统运行在 **Microsoft** 基于 **Azure** 的超级计算平台上。
>
> [OpenAI API](https://platform.openai.com/docs/models) 由一套拥有不同功能和价格点的模型提供支持。
> 
> [ChatGPT](https://chat.openai.com) 是由 `OpenAI` 开发的人工智能（AI）聊天机器人。

## 安装和设置

使用以下命令安装集成包
```bash
pip install langchain-openai
```

获取OpenAI API密钥，并将其设置为环境变量（`OPENAI_API_KEY`）

## Chat模型

请参阅[使用示例](/docs/integrations/chat/openai)。

```python
from langchain_openai import ChatOpenAI
```

如果您使用的是托管在 `Azure` 上的模型，则应使用不同的包装器：
```python
from langchain_openai import AzureChatOpenAI
```
有关 `Azure` 包装器的更详细演练，请参阅[此处](/docs/integrations/chat/azure_chat_openai)。

### Dall-E图像生成器

>[OpenAI Dall-E](https://openai.com/dall-e-3) 是由 `OpenAI` 开发的文本到图像模型，
>它使用深度学习方法，根据自然语言描述（称为“提示”）生成数字图像。


请参阅[使用示例](/docs/integrations/tools/dalle_image_generator)。

```python

```

## Tokenizer

您可以在多个地方使用 `tiktoken` 分词器。默认情况下，它用于计算OpenAI LLM的token数。

您也可以在分割文档时使用它：
```python
from langchain.text_splitter import CharacterTextSplitter
CharacterTextSplitter.from_tiktoken_encoder(...)
```
有关此内容的更详细演练，请参阅[此笔记本](/docs/how_to/split_by_token/#tiktoken)
