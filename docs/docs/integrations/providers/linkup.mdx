# Linkup

> [Linkup](https://www.linkup.so/) 提供了一个 API，用于将 LLM 连接到网络和 Linkup Premium Partner 资源。

## 安装和设置

要使用 Linkup 提供程序，您首先需要一个有效的 API 密钥，您可以通过在此处 [注册](https://app.linkup.so/sign-up) 找到。
您还需要 `langchain-linkup` 包，可以使用 pip 安装：

```bash
pip install langchain-linkup
```

## Retriever

请参阅 [使用示例](/docs/integrations/retrievers/linkup_search)。

```python
from langchain_linkup import LinkupSearchRetriever

retriever = LinkupSearchRetriever(
    depth="deep",  # "standard" 或 "deep"
    linkup_api_key=None,  # API 密钥可以在此处传递，或设置为 LINKUP_API_KEY 环境变量
)
```

## Tools

请参阅 [使用示例](/docs/integrations/tools/linkup_search)。

```python
from langchain_linkup import LinkupSearchTool

tool = LinkupSearchTool(
    depth="deep",  # "standard" 或 "deep"
    output_type="searchResults",  # "searchResults", "sourcedAnswer" 或 "structured"
    linkup_api_key=None,  # API 密钥可以在此处传递，或设置为 LINKUP_API_KEY 环境变量
)
```
