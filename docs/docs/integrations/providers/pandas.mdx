# Pandas

>[pandas](https://pandas.pydata.org) 是一个快速、强大、灵活且易于使用的开源数据分析和操作工具，
构建在 `Python` 编程语言之上。

## 安装和设置

使用 `pip` 安装 `pandas` 包：

```bash
pip install pandas
```
