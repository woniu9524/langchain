# Dria

>[Dria](https://dria.co/) 是一个面向开发者的公共 RAG 模型中心，开发者可以在此贡献和利用共享的嵌入湖。

在[此页面](https://dria.co/docs/integrations/langchain)上可以查看 LangChain 与 Dria 集成的更多详细信息。

## 安装和设置

您需要安装一个 Python 包：

```bash
pip install dria
```

您需要从 Dria 获取一个 API 密钥。您可以通过在[Dria](https://dria.co/) 注册来获取。
