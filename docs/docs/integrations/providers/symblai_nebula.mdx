# Nebula

本页介绍如何在 LangChain 中使用 [Nebula](https://symbl.ai/nebula)，即 [Symbl.ai](https://symbl.ai/) 的 LLM 生态系统。
本页分为两部分：安装和设置，以及指向特定 Nebula 包装器的参考。

## 安装和设置

- 获取 [Nebula API Key](https://info.symbl.ai/Nebula_Private_Beta.html) 并将其设置为环境变量 `NEBULA_API_KEY`
- 有关更多详细信息，请参阅 [Nebula 文档](https://docs.symbl.ai/docs/nebula-llm)。

### LLM

存在 Nebula LLM 包装器，您可以通过以下方式访问它：
```python
from langchain_community.llms import Nebula
llm = Nebula()
```
