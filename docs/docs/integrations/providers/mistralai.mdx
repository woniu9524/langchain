# MistralAI

>[Mistral AI](https://docs.mistral.ai/api/) 是一个提供强大开源模型托管的平台。

## 安装与设置

需要一个有效的 [API 密钥](https://console.mistral.ai/users/api-keys/) 才能与 API 通信。

您还需要 `langchain-mistralai` 包：

```bash
pip install langchain-mistralai
```

## Chat 模型

### ChatMistralAI

请参阅 [使用示例](/docs/integrations/chat/mistralai)。

```python
from langchain_mistralai.chat_models import ChatMistralAI
```

## Embedding 模型

### MistralAIEmbeddings

请参阅 [使用示例](/docs/integrations/text_embedding/mistralai)。

```python
from langchain_mistralai import MistralAIEmbeddings
```
