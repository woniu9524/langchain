# LiteLLM

>[LiteLLM](https://github.com/BerriAI/litellm) 是一个简化调用 Anthropic、Azure、Huggingface、Replicate 等库的库。

## 安装和设置

```bash
pip install langchain-litellm
```

## Chat Models
```python
from langchain_litellm import ChatLiteLLM
```
```python
from langchain_litellm import ChatLiteLLMRouter
```
更多详情请参阅指南 [此处](/docs/integrations/chat/litellm)。

## API 参考
有关 `ChatLiteLLM` 和 `ChatLiteLLMRouter` 所有功能和配置的详细文档，请访问 API 参考：https://github.com/Akshay-Dongare/langchain-litellm
