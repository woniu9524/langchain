# vlite

本页介绍如何在 LangChain 中使用 [vlite](https://github.com/sdan/vlite)。vlite 是一个简单、快速的向量数据库，用于存储和检索嵌入。

## 安装和设置

要安装 vlite，请运行以下命令：

```bash
pip install vlite
```

如需 PDF OCR 支持，请安装 `vlite[ocr]` 附加组件：

```bash
pip install vlite[ocr]
```

## VectorStore

vlite 提供了一个围绕其向量数据库的封装器，允许您将其用作向量存储，用于语义搜索和示例选择。

要导入 vlite vectorstore：

```python
from langchain_community.vectorstores import vlite
```
