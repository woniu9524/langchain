# Golden

>[Golden](https://golden.com) 提供了一组自然语言 API，用于通过 Golden Knowledge Graph 进行查询和丰富，例如：`Products from OpenAI`、`Generative ai companies with series a funding` 和 `rappers who invest` 等查询可用于检索关于相关实体的结构化数据。
>
>`golden-query` langchain 工具是对 [Golden Query API](https://docs.golden.com/reference/query-api) 的封装，它能够以编程方式访问这些结果。
>有关更多信息，请参阅 [Golden Query API 文档](https://docs.golden.com/reference/query-api)。

## 安装和设置
- 请访问 [Golden API 文档](https://docs.golden.com/) 了解 Golden API 的概述。
- 从 [Golden API 设置](https://golden.com/settings/api) 页面获取您的 API 密钥。
- 将您的 API 密钥保存到 GOLDEN\_API\_KEY 环境变量中。

## 封装器

### 工具类

存在一个 `GoldenQueryAPIWrapper` 工具类来封装此 API。导入此工具类：

```python
from langchain_community.utilities.golden_query import GoldenQueryAPIWrapper
```

有关该工具类的更详细的演练，请参阅 [此笔记本](/docs/integrations/tools/golden_query)。

### 工具

您也可以轻松地将此工具类加载为 Tool（用于 Agent）。
您可以通过以下方式实现：
```python
from langchain_community.agent_toolkits.load_tools import load_tools
tools = load_tools(["golden-query"])
```

有关工具的更多信息，请参阅 [此页面](/docs/how_to/tools_builtin)。
