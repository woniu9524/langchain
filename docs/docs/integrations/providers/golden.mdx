# Golden

>[Golden](https://golden.com) 提供了一组自然语言 API，用于使用 Golden Knowledge Graph 进行查询和丰富，例如：`Products from OpenAI`、`Generative ai companies with series a funding` 和 `rappers who invest` 等查询可用于检索相关实体的结构化数据。
>
> `golden-query` langchain 工具是 [Golden Query API](https://docs.golden.com/reference/query-api) 之上的一个封装，它支持对这些结果进行编程访问。
> 有关更多信息，请参阅 [Golden Query API 文档](https://docs.golden.com/reference/query-api)。

## 安装和设置
- 前往 [Golden API 文档](https://docs.golden.com/) 了解 Golden API 的概述。
- 从 [Golden API 设置](https://golden.com/settings/api) 页面获取您的 API 密钥。
- 将您的 API 密钥保存到 GOLDEN_API_KEY 环境变量中。

## 封装

### 工具类

存在一个名为 GoldenQueryAPIWrapper 的工具类，它封装了这个 API。要导入这个工具类：

```python
from langchain_community.utilities.golden_query import GoldenQueryAPIWrapper
```

有关此工具类的更详细的教程，请参阅 [此笔记本](/docs/integrations/tools/golden_query)。

### 工具

您也可以轻松地将此工具类加载为工具（以与 Agent 一起使用）。
您可以通过以下方式实现：
```python
from langchain_community.agent_toolkits.load_tools import load_tools
tools = load_tools(["golden-query"])
```

有关工具的更多信息，请参阅 [此页面](/docs/how_to/tools_builtin)。
