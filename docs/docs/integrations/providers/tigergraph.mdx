# TigerGraph

>[TigerGraph](https://www.tigergraph.com/tigergraph-db/) 是一个原生分布式高性能图数据库。
> 以顶点和边的图格式存储数据，可以形成丰富的关系，非常适合为大语言模型响应提供基础。

## 安装和设置

请遵循[如何连接到 `TigerGraph` 数据库](https://docs.tigergraph.com/pytigergraph/current/getting-started/connection)的说明。

安装 Python SDK：

```bash
pip install pyTigerGraph
```

## 图存储

### TigerGraph

请参阅[用法示例](/docs/integrations/graphs/tigergraph)。

```python
from langchain_community.graphs import TigerGraph
```
