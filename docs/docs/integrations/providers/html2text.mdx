# HTML 转文本

>[html2text](https://github.com/Alir3z4/html2text/) 是一个 Python 包，可以将页面上的 `HTML` 转换为干净、易读的纯 `ASCII` 文本。

该 `ASCII` 文本恰好也是有效的 `Markdown`（一种文本到 HTML 的格式）。

## 安装和设置

```bash
pip install html2text
```

## Document Transformer

请参阅[用法示例](/docs/integrations/document_transformers/html2text)。

```python
from langchain_community.document_loaders import Html2TextTransformer
```
