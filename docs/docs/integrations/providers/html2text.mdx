# HTML 转文本

>[html2text](https://github.com/Alir3z4/html2text/) 是一个 Python 包，可以将一个 `HTML` 页面转换为干净、易于阅读的纯 `ASCII` 文本。

ASCII 文本恰好也是有效的 `Markdown`（一种文本到 HTML 的格式）。

## 安装和设置

```bash
pip install html2text
```

## 文档转换器

请参阅 [使用示例](/docs/integrations/document_transformers/html2text)。

```python
from langchain_community.document_loaders import Html2TextTransformer
