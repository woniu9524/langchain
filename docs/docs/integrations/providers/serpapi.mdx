# SerpAPI

此页面涵盖了如何在 LangChain 中使用 SerpAPI 搜索 API。
它分为两部分：安装和设置，以及对特定 SerpAPI 封装器的引用。

## 安装与设置
- 使用 `pip install google-search-results` 安装所需库
- 获取 SerpAPI API 密钥，并将其设置为环境变量 (`SERPAPI_API_KEY`)

## 封装器

### 工具类

存在一个名为 SerpAPI 的工具类封装了此 API。导入此工具类：

```python
from langchain_community.utilities import SerpAPIWrapper
```

有关此封装器的更详细介绍，请参阅[此笔记本](/docs/integrations/tools/serpapi)。

### 工具集

您也可以轻松地将此封装器加载为工具集（与 Agent 一起使用）。
方法如下：
```python
from langchain_community.agent_tool_kits.load_tools import load_tools
tools = load_tools(["serpapi"])
```

有关此内容的更多信息，请参阅[此页面](/docs/how_to/tools_builtin)
