# KDB.AI

>[KDB.AI](https://kdb.ai) 是一个强大的知识库向量数据库和搜索引擎，它通过提供高级搜索、推荐和个性化功能，使您能够使用实时数据构建可扩展、可靠的 AI 应用程序。


## 安装和设置

安装 Python SDK：

```bash
pip install kdbai-client
```


## Vector store

存在一个围绕 KDB.AI 索引的封装，允许您将其用作向量存储，无论是用于语义搜索还是示例选择。

```python
from langchain_community.vectorstores import KDBAI
```

有关 KDB.AI 向量存储的更详细的演练，请参阅[此笔记本](/docs/integrations/vectorstores/kdbai)
