# Apache 软件基金会

>[Apache软件基金会（维基百科）](https://en.wikipedia.org/wiki/The_Apache_Software_Foundation)
>是一个去中心化的开源社区。他们开发的软件根据 Apache许可证发布，这是一个对自由开源软件（FOSS）宽松的开源许可证。Apache项目以协作、共识驱动的开发流程，以及开放务实的软件许可证为特点，这意味着它允许免费获取软件的开发者以非免费条款重新分发它。每个项目都由一个由活跃贡献者组成的、自行选拔的技术专家团队管理。

## Apache AGE

>[Apache AGE](https://age.apache.org/) 是一个提供图数据库功能的 `PostgreSQL` 扩展。`AGE` 是 `A Graph Extension` 的缩写，灵感来源于 Bitnine 对 `PostgreSQL 10` 的分支 `AgensGraph`，它是一个多模数据库。该项目的目标是创建一个可以同时处理关系模型和图模型数据的单一存储，以便用户可以使用标准的 ANSI SQL 和图查询语言 `openCypher`。`Apache AGE` 存储的数据元素包括节点、连接它们的边，以及节点和边的属性。

详细了解 [与 Apache AGE 集成](/docs/integrations/graphs/apache_age)。

## Apache Cassandra

>[Apache Cassandra](https://cassandra.apache.org/) 是一个 NoSQL、面向行、高度可扩展且高可用的数据库。从 5.0 版本开始，该数据库提供了向量搜索功能。

详细了解 [与 Apache Cassandra 集成](/docs/integrations/providers/cassandra/)。

## Apache Doris

>[Apache Doris](https://doris.apache.org/) 是一个用于实时分析的现代数据仓库。它能在规模化数据上为实时数据提供闪电般的分析速度。
>
>通常 `Apache Doris` 被归类为 OLAP，并且在分析型数据库基准测试 ClickBench 中表现出色。由于它拥有超快的向量化执行引擎，也可以用作快速的向量数据库。

详细了解 [与 Apache Doris 集成](/docs/integrations/providers/apache_doris/)。

## Apache Kafka

>[Apache Kafka](https://github.com/apache/kafka) 是一个分布式消息系统，用于发布和订阅记录流。

详细了解 [与 Apache Kafka 集成](/docs/integrations/memory/kafka_chat_message_history)。

## Apache Spark

>[Apache Spark](https://spark.apache.org/) 是一个用于大规模数据处理的统一分析引擎。它提供了 Scala、Java、Python 和 R 的高级 API，以及一个支持数据分析通用计算图的优化引擎。它还支持丰富的工具集，包括用于 SQL 和 DataFrame 的 `Spark SQL`，用于 pandas 工作负载的 `pandas API on Spark`，用于机器学习的 `MLlib`，用于图处理的 `GraphX`，以及用于流处理的 `Structured Streaming`。

详细了解 [与 Apache Spark 集成](/docs/integrations/providers/spark)。
