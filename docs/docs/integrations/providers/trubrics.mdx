# Trubrics


>[Trubrics](https://trubrics.com) 是一个 LLM 用户分析平台，可让您收集、分析和管理 AI 模型的用户提示和反馈。

>有关 `Trubrics` 的更多信息，请查看 [Trubrics 仓库](https://github.com/trubrics/trubrics-sdk)。

## 安装和设置

我们需要安装 `trubrics` Python 包：

```bash
pip install trubrics
```


## 回调

请参阅 [用法示例](/docs/integrations/callbacks/trubrics)。

```python
from langchain.callbacks import TrubricsCallbackHandler
```
