# KoboldAI

>[KoboldAI](https://koboldai.com/) 是一个免费的开源项目，允许用户在自己的计算机上本地运行 AI 模型。
>它是一个基于浏览器的前端，可用于与 AI 进行写作或角色扮演。
>[KoboldAI](https://github.com/KoboldAI/KoboldAI-Client) 是一个“一个基于浏览器的前端，用于 AI 辅助写作，支持多个本地和远程 AI 模型……”。
>它有一个公共和本地 API，可以在 LangChain 中使用。
