# Cloudflare

>[Cloudflare, Inc. (Wikipedia)](https://en.wikipedia.org/wiki/Cloudflare)是一家美国公司，提供
> 内容分发网络服务、云安全、DDoS缓解和ICANN认可的
> 域名注册服务。

>[Cloudflare Workers AI](https://developers.cloudflare.com/workers-ai/)允许您运行机器学习
> 模型，在`Cloudflare`网络上，通过REST API从您的代码中运行。
