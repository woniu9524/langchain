# Cloudflare

>[Cloudflare, Inc. (Wikipedia)](https://en.wikipedia.org/wiki/Cloudflare) 是一家美国公司，提供内容分发网络服务、云安全、DDoS 防护以及 ICANN 认证的域名注册服务。

>[Cloudflare Workers AI](https://developers.cloudflare.com/workers-ai/) 允许您通过 REST API 在 `Cloudflare` 网络上从代码中运行机器学习模型。
