# Vearch

[Vearch](https://github.com/vearch/vearch) 是一个可扩展的分布式系统，用于高效地对深度学习向量进行相似性搜索。

# 安装和设置

Vearch Python SDK 可以方便地在本地使用 Vearch。可以通过 `pip install vearch` 轻松安装 Vearch Python SDK。

# Vectorstore

Vearch 也可用作 vectorstore。更多细节请参见[此笔记本](/docs/integrations/vectorstores/vearch)

from langchain_community.vectorstores import Vearch
