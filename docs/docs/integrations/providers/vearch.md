# Vearch

[Vearch](https://github.com/vearch/vearch) 是一个可扩展的分布式系统，用于高效地进行深度学习向量的相似性搜索。

# 安装和设置

Vearch Python SDK 使您可以在本地使用 Vearch。通过 pip install vearch 可以轻松安装 Vearch python sdk。
