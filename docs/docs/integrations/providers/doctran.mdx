# Doctran

>[Doctran](https://github.com/psychic-api/doctran) 是一个 Python 包。它使用 LLM 和开源 NLP 库将原始文本转换为干净、结构化、信息密集型的文档，以优化向量空间检索。您可以将 `Doctran` 视为一个黑匣子，混乱的字符串进入，整洁、干净、带标签的字符串出来。


## 安装和设置

```bash
pip install doctran
```
