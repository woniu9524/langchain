# Git

>[Git](https://en.wikipedia.org/wiki/Git) 是一个分布式版本控制系统，用于跟踪任何一组计算机文件中的更改，通常用于在软件开发过程中协调程序员协同开发源代码的工作。

## 安装与设置

首先，您需要安装 `GitPython` Python 包。

```bash
pip install GitPython
```
