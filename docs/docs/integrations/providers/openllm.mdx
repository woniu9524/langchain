# OpenLLM

OpenLLM 允许开发者通过**一个命令**将任何**开源 LLM** 作为**兼容 OpenAI 的 API** 端点来运行。

- 🔬 为快速和生产环境构建
- 🚂 支持 llama3、qwen2、gemma 等，以及许多**量化**版本 [完整列表](https://github.com/bentoml/openllm-models)
- ⛓️ 兼容 OpenAI 的 API
- 💬 内置类似 ChatGPT 的 UI
- 🔥 使用最先进的推理后端加速 LLM 解码
- 🌥️ 准备好进行企业级云部署（Kubernetes、Docker 和 BentoCloud）

## 安装和设置

通过 PyPI 安装 OpenLLM 包：

```bash
pip install openllm
```

## LLM

OpenLLM 支持各种开源 LLM 以及用户自己的微调 LLM。使用 `openllm model` 命令查看所有可用的、为 OpenLLM 预优化的模型。

## 包装器

有一个 OpenLLM 包装器支持与运行中的 OpenLLM 服务器进行交互：

```python
from langchain_community.llms import OpenLLM
```

### OpenLLM 服务器包装器

此包装器支持与 OpenLLM 的兼容 OpenAI 端点进行交互。

要运行模型，请执行：

```bash
openllm hello
```

包装器用法：

```python
from langchain_community.llms import OpenLLM

llm = OpenLLM(base_url="http://localhost:3000/v1", api_key="na")

llm("What is the difference between a duck and a goose? And why there are so many Goose in Canada?")
```
