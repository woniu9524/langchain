# YouTube

>[YouTube](https://www.youtube.com/) 是由谷歌推出的在线视频分享和社交媒体平台。
> 我们下载 `YouTube` 的视频字幕和视频信息。

## 安装与设置

```bash
pip install youtube-transcript-api
pip install pytube
```
请参阅[使用示例](/docs/integrations/document_loaders/youtube_transcript)。
