# Aerospike

>[Aerospike](https://aerospike.com/docs/vector) 是一个高性能的分布式数据库，以其速度和可扩展性而闻名，现在支持向量存储和搜索，能够为机器学习和人工智能应用检索和搜索嵌入向量。
> 在[此处](https://aerospike.com/docs/vector)查看 Aerospike Vector Search (AVS) 的文档。

## 安装和设置

安装 AVS Python SDK 和 AVS langchain vector store：

```bash
pip install aerospike-vector-search langchain-aerospike
```

在[此处](https://aerospike-vector-search-python-client.readthedocs.io/en/latest/index.html)查看 Python SDK 的文档。
AVS langchain vector store 的文档在[此处](https://langchain-aerospike.readthedocs.io/en/latest/)。

## Vector Store

要导入此 vectorstore：

```python
from langchain_aerospike.vectorstores import Aerospike
```

查看使用示例[此处](https://python.langchain.com/docs/integrations/vectorstores/aerospike/)。
