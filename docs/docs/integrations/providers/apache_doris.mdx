# Apache Doris

>[Apache Doris](https://doris.apache.org/) 是一个用于实时分析的现代数据仓库。
>它能够在大规模数据集上对实时数据进行闪电般的快速分析。

>通常 `Apache Doris` 被归类为 OLAP，并且在
>[ClickBench — a Benchmark For Analytical DBMS](https://benchmark.clickhouse.com/) 中表现出色。
>由于它拥有超快的向量化执行引擎，也可以用作快速的向量数据库。

## 安装和设置

```bash
pip install pymysql
```
