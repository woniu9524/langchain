# SAP

>[SAP SE(维基百科)](https://www.sap.com/about/company.html) 是一家德国跨国软件公司。它开发企业软件来管理业务运营和客户关系。该公司是世界领先的 `企业资源规划 (ERP)` 软件供应商。

## 安装和设置

我们需要安装 `langchain-hana` Python 包。

```bash
pip install langchain-hana
```
