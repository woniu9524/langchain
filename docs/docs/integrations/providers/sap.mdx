# SAP

>[SAP SE(Wikipedia)](https://www.sap.com/about/company.html) 是一家德国跨国软件公司。它开发用于管理业务运营和客户关系的商用软件。该公司是全球领先的 `企业资源规划 (ERP)` 软件供应商。

## 安装与设置

我们需要安装 `langchain-hana` Python 包。

```bash
pip install langchain-hana
```
