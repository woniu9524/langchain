# USearch
>[USearch](https://unum-cloud.github.io/usearch/) 是一个更小、更快的单文件向量搜索引擎。

>`USearch` 的核心功能与 `FAISS` 相同，如果您曾研究过近似最近邻搜索，其接口应该看起来很熟悉。
> `USearch` 和 `FAISS` 都采用了 `HNSW` 算法，但在设计原则上存在显著差异。
>`USearch` 精巧且与 FAISS 广泛兼容，同时不牺牲性能，其主要侧重于用户定义的指标和更少的依赖项。
## 安装和设置

我们需要安装 `usearch` Python 包。

```bash
pip install usearch
```
