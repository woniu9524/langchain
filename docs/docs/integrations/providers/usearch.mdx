# USearch
>[USearch](https://unum-cloud.github.io/usearch/) 是一个更小、更快、单文件的向量搜索引擎。

>`USearch` 的核心功能与 `FAISS` 相同，如果您曾研究过近似最近邻搜索（Approximate Nearest Neighbors），其接口应该很熟悉。
>`USearch` and `FAISS` 都采用了 `HNSW` 算法，但在设计原则上存在显著差异。
>`USearch` 设计紧凑，与 FAISS 兼容性好，且不牺牲性能，主要侧重于用户自定义指标和减少依赖。
## 安装与设置

我们需要安装 `usearch` Python 包。

```bash
pip install usearch
```
