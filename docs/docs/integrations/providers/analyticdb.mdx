# AnalyticDB

>[AnalyticDB for PostgreSQL](https://www.alibabacloud.com/help/en/analyticdb-for-postgresql/latest/product-introduction-overview) 
> 是来自[阿里云](https://www.alibabacloud.com/)的**海量并行处理**（MPP）数据仓库服务，
> 旨在在线分析海量数据。

>`AnalyticDB for PostgreSQL` 基于开源的 `Greenplum Database` 
> 项目开发，并经过阿里云的深度扩展。AnalyticDB 
> for PostgreSQL 支持 ANSI SQL 2003 语法，并与 PostgreSQL 和 
> Oracle 数据库生态系统兼容。AnalyticDB for PostgreSQL 同时支持行存储和
> 列存储。AnalyticDB for PostgreSQL 以高性能处理 PB 级别的数据离线分析，并支持高并发。

本页面介绍如何在 LangChain 中使用 AnalyticDB 生态系统。

## 安装和设置

您需要安装 `sqlalchemy` python 包。

```bash
pip install sqlalchemy
```
