# Pebblo

[Pebblo](https://www.daxa.ai/pebblo) 使开发人员能够安全地加载和检索数据，从而在无需担心组织的合规性和安全要求的情况下，将其 Gen AI 应用推广到部署阶段。Pebblo SafeLoader 可识别加载数据中存在的语义主题和实体，而 Pebblo SafeRetriever 则在检索到的上下文中强制执行身份和语义控制。结果会在 UI 或 PDF 报告中进行总结。

## Pebblo 概述：

`Pebblo` 为 Gen AI 应用提供了一种安全的数据加载和检索方式。
它包括：
1. **身份感知安全加载器**，用于加载数据并识别语义主题和实体。
2. **安全检索**，用于在检索到的上下文中强制执行身份和语义控制。
3. **用户数据报告**，用于总结加载和检索的数据。
