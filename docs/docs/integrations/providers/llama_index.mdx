# LlamaIndex

>[LlamaIndex](https://www.llamaindex.ai/) 是构建 LLM 应用程序的领先数据框架。

## 安装和设置

您需要安装 `llama-index` Python 包。

```bash
pip install llama-index
```

请参阅[安装说明](https://docs.llamaindex.ai/en/stable/getting_started/installation/)。

## Retrievers

### LlamaIndexRetriever

>它用于对 LlamaIndex 数据结构进行带来源的问答。

```python
from langchain_community.retrievers.llama_index import LlamaIndexRetriever
```

### LlamaIndexGraphRetriever

>它用于对 LlamaIndex 图数据结构进行带来源的问答。

```python
from langchain_community.retrievers.llama_index import LlamaIndexGraphRetriever
