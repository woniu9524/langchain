# Neo4j

>什么是 `Neo4j`？

>- Neo4j 是一个`开源数据库管理系统`，专注于图数据库技术。
>- Neo4j 允许您以节点和边的方式表示和存储数据，非常适合处理连接数据和关系。
>- Neo4j 提供了一个`Cypher 查询语言`，方便您与图数据进行交互和查询。
>- 使用 Neo4j，您可以实现高性能的`图遍历和查询`，适用于生产级系统。

>访问 [他们的网站](https://neo4j.com/) 开始了解 Neo4j。

## 安装和设置

- 使用 `pip install neo4j langchain-neo4j` 安装 Python SDK

## VectorStore

Neo4j 向量索引用作向量存储，用于语义搜索或示例选择。

```python
from langchain_neo4j import Neo4jVector
```

查看[用法示例](/docs/integrations/vectorstores/neo4jvector)

## GraphCypherQAChain

Neo4j 图数据库提供了一个包装器，允许您根据用户输入生成 Cypher 语句，并使用它们从数据库检索相关信息。

```python
from langchain_neo4j import GraphCypherQAChain, Neo4jGraph
```

查看[用法示例](/docs/integrations/graphs/neo4j_cypher)

## 从文本构建知识图

文本数据通常包含丰富的关系和见解，可用于各种分析、推荐引擎或知识管理应用程序。
Diffbot 的 NLP API 可以从非结构化文本数据中提取实体、关系和语义。
通过结合 Diffbot 的 NLP API 和图数据库 Neo4j，您可以根据从文本中提取的信息创建强大、动态的图结构。
这些图结构完全可查询，并可以集成到各种应用程序中。

```python
from langchain_neo4j import Neo4jGraph
from langchain_experimental.graph_transformers.diffbot import DiffbotGraphTransformer
```

查看[用法示例](/docs/integrations/graphs/diffbot)
