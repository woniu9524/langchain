# Writer, Inc.

All functionality related to Writer


>本页面介绍了如何在 LangChain 中使用 Writer 生态系统。有关更多信息，请参阅 Writer [文档](https://dev.writer.com/home/introduction)。
>[Palmyra](https://writer.com/blog/palmyra/) 是由 `Writer, Inc` 开发的大型语言模型 (LLM)。
>
> [Writer API](https://dev.writer.com/api-guides/introduction) 由一套多样化的 Palmyra 子模型提供支持，这些子模型具有不同的功能和价格点。

## 安装与设置

使用以下命令安装集成包
```bash
pip install langchain-writer
```

获取 Writer API 密钥并将其设置为环境变量 (`WRITER_API_KEY`)

## Chat 模型

```python
from langchain_writer import ChatWriter
```
请参阅 [详情](/docs/integrations/chat/writer)。

## PDF 解析器



```python

```
请参阅 [详情](/docs/integrations/splitters/writer_text_splitter)。
