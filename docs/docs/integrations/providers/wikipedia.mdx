# Wikipedia

>[Wikipedia](https://wikipedia.org/) 是一个多语言的免费在线百科全书，由称为维基百科人的社区志愿者通过开放协作和使用名为 MediaWiki 的维基编辑系统编写和维护。`Wikipedia` 是历史上最大、阅读量最多的参考资料。


## 安装和设置

```bash
pip install wikipedia
```



## 文档加载器

请参阅[用法示例](/docs/integrations/document_loaders/wikipedia)。

```python
from langchain_community.document_loaders import WikipediaLoader
```
