# Wikipedia

>[Wikipedia](https://wikipedia.org/) 是一个多语言的免费在线百科全书，由称为“维基百科人”的志愿者社区通过开放协作编写和维护，并使用名为 MediaWiki 的基于 wiki 的编辑系统。`Wikipedia` 是历史上最大、阅读量最多的参考资料。


## 安装和设置

```bash
pip install wikipedia
```



## Document Loader

请参阅[用法示例](/docs/integrations/document_loaders/wikipedia)。

```python
from langchain_community.document_loaders import WikipediaLoader
```
