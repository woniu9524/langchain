# Xata

> [Xata](https://xata.io) 是一个无服务器数据平台，基于 `PostgreSQL` 构建。
> 它提供了一个 Python SDK 用于与你的数据库交互，并提供了一个 UI
> 用于管理你的数据。
> `Xata` 有一个原生的向量类型，可以添加到任何表中，并支持相似性搜索。LangChain 直接将向量插入 `Xata`，
> 并查询给定向量的最近邻，以便你能将所有 LangChain Embeddings 集成与 `Xata` 一起使用。

## 安装和设置

我们需要安装 `xata` Python 包。

```bash
pip install xata==1.0.0a7 
```
