# Dell

Dell 是一家全球性科技公司，提供一系列硬件、软件和服务，其中包括人工智能解决方案。他们的人工智能产品组合包括专为人工智能工作负载设计的专用基础设施，例如 Dell PowerScale 存储系统，该系统针对人工智能数据管理进行了优化。

## PowerScale

Dell [PowerScale](https://www.dell.com/en-us/shop/powerscale-family/sf/powerscale) 是
一个企业级横向扩展存储系统，托管着行业领先的 OneFS 文件系统，可以部署在本地或云端。

### 安装和设置

```bash
pip install powerscale-rag-connector
```
