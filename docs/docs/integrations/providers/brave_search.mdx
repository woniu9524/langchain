# Brave Search


>[Brave Search](https://en.wikipedia.org/wiki/Brave_Search) 是 Brave Software 开发的一款搜索引擎。
> - `Brave Search` 使用自己的网页索引。截至 2022 年 5 月，它覆盖了超过 100 亿个页面，并且 92% 的搜索结果无需依赖任何第三方即可提供，其余结果从 Bing API 服务器端检索，或（选择加入的基础上）从 Google 客户端检索。根据 Brave 的说法，为了帮助避免垃圾邮件和其他低质量内容，该索引“特意比 Google 或 Bing 的索引小”，缺点是“Brave Search 在恢复长尾查询方面还不如 Google”。
>- `Brave Search Premium`：截至 2023 年 4 月，Brave Search 是一个无广告网站，但它最终将切换到一个包含广告的新模式，付费用户将获得无广告体验。
> 默认情况下，用户数据（包括 IP 地址）将不会从用户那里收集。需要付费账户才能选择加入数据收集。


## 安装和设置

要访问 Brave Search API，您需要[创建一个账户并获取 API 密钥](https://api.search.brave.com/app/dashboard)。
