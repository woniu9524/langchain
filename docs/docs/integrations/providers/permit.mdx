# Permit

[Permit.io](https://permit.io/) 提供细粒度的访问控制和策略执行。通过 LangChain，您可以集成 Permit 检查，确保只有授权用户才能访问或检索您 LLM 应用中的数据。

## 安装和设置

```bash
pip install langchain-permit
pip install permit
```

为您的 Permit PDP 和凭据设置环境变量：

```python
export PERMIT_API_KEY="your_permit_api_key"
export PERMIT_PDP_URL="http://localhost:7766"   # 或者您的真实 PDP 端点
```

请确保您的 PDP 正在运行并已配置。有关策略设置，请参阅
[Permit 文档](https://docs.permit.io/sdk/python/quickstart-python/#2-setup-your-pdp-policy-decision-point-container)。
