# Infinispan VS

> [Infinispan](https://infinispan.org) Infinispan 是一个开源的内存数据网格，提供了一个键/值数据存储，能够容纳从 Java 对象到纯文本的所有类型数据。
> 从 15 版本开始，Infinispan 支持在缓存上进行向量搜索。

## 安装和设置
请参阅 [入门](https://infinispan.org/get-started/) 来运行 Infinispan 服务器，您可能希望禁用身份验证
（目前不支持）
