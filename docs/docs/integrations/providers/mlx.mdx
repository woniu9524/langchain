# MLX

>[MLX](https://ml-explore.github.io/mlx/build/html/index.html) 是一个 `NumPy`-风格的数组框架，专为在 `Apple` 芯片上进行高效且灵活的机器学习而设计，由 `Apple machine learning research` 提供。


## 安装与设置

安装几个 Python 包：

```bash
pip install mlx-lm transformers huggingface_hub
````


## Chat models


查看 [使用示例](/docs/integrations/chat/mlx)。

```python
from langchain_community.chat_models.mlx import ChatMLX
```
