# Streamlit

>[Streamlit](https://streamlit.io/) 是构建和共享数据应用的一种更快捷的方式。
>`Streamlit` 能在几分钟内将数据脚本转换为可共享的 Web 应用。全部使用纯 Python。无需前端经验。
>在 [streamlit.io/generative-ai](https://streamlit.io/generative-ai) 查看更多示例。

## 安装和设置

我们需要安装 `streamlit` Python 包：

```bash
pip install streamlit
```

## 内存

请参阅 [用法示例](/docs/integrations/memory/streamlit_chat_message_history)。

```python
from langchain_community.chat_message_histories import StreamlitChatMessageHistory
```
