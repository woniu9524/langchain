# BibTeX

>[BibTeX](https://www.ctan.org/pkg/bibtex) 是一种文件格式和参考文献管理系统，通常与 `LaTeX` 排版系统结合使用。它是一种组织和存储学术及研究文献作品的参考文献信息的方式。

## 安装和设置

我们需要安装 `bibtexparser` 和 `pymupdf` 这两个包。

```bash
pip install bibtexparser pymupdf
```

## 文档加载器

请参阅[用法示例](/docs/integrations/document_loaders/bibtex)。

```python
from langchain_community.document_loaders import BibtexLoader
```
