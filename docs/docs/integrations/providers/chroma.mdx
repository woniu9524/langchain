# Chroma

>[Chroma](https://docs.trychroma.com/getting-started) 是一个用于构建具有嵌入式 AI 应用程序的数据库。

## 安装与设置

```bash
pip install langchain-chroma
```

## VectorStore

Chroma vector 数据库提供了一个封装，允许您将其用作 vectorstore，无论是用于语义搜索还是示例选择。

```python
from langchain_chroma import Chroma
```

有关 Chroma 封装的更详细的教程，请参阅[此笔记本](/docs/integrations/vectorstores/chroma)。
