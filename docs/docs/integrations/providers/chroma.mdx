# Chroma

>[Chroma](https://docs.trychroma.com/getting-started) 是一个用于构建 AI 应用程序的数据库，支持 embeddings。

## 安装与设置

```bash
pip install langchain-chroma
```


## VectorStore

有一个 Chroma 向量数据库的封装，允许你将其用作 vectorstore，用于语义搜索或示例选择。

```python
from langchain_chroma import Chroma
```

要更详细地了解 Chroma 封装，请参阅[此 notebook](/docs/integrations/vectorstores/chroma)。
