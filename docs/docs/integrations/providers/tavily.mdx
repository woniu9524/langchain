# Tavily

[Tavily](https://tavily.com) 是一个搜索引擎，专为 AI 代理设计。
Tavily 提供搜索和提取 API，AI 开发人员可以轻松地将其应用程序与实时在线信息集成。Tavily 的主要任务是从可信赖的来源提供事实性和可靠信息，从而提高 AI 生成内容和推理的准确性和可靠性。

## 安装和设置

```bash
pip install langchain-tavily
```
