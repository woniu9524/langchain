# Meilisearch

> [Meilisearch](https://meilisearch.com) 是一个开源的、闪电般快速且高度相关的搜索引擎。
> 它带有一系列优秀的默认设置，可帮助开发人员构建响应迅速的搜索体验。
>
> 您可以[自行托管 Meilisearch](https://www.meilisearch.com/docs/learn/getting_started/installation#local-installation) 或在 [Meilisearch Cloud](https://www.meilisearch.com/pricing) 上运行。
>
>`Meilisearch v1.3` 支持向量搜索。

## 安装与设置

请参阅 [使用示例](/docs/integrations/vectorstores/meilisearch) 获取详细配置说明。

我们需要安装 `meilisearch` Python 包。

```bash
pip install meilisearch
```
