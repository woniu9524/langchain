# StarRocks

>[StarRocks](https://www.starrocks.io/) 是一个高性能分析数据库。
`StarRocks` 是下一代亚秒级 MPP 数据库，适用于全分析场景，包括多维分析、实时分析和即席查询。

>通常 `StarRocks` 被归类为 OLAP，在[ClickBench — 面向分析型数据库的基准测试](https://benchmark.clickhouse.com/)中表现出色。由于它拥有超快的向量化执行引擎，也可以用作快速向量数据库。

## 安装与设置

```bash
pip install pymysql
```
