# Cube

>[Cube](https://cube.dev/) 是用于构建数据应用的语义层。它帮助数据工程师和应用程序开发者从现代数据存储访问数据，将数据组织成一致的定义，并将数据交付到每个应用程序。

## 安装与设置

我们需要获取 API 密钥和 Cube 实例的 URL。请参阅
[这些说明](https://cube.dev/docs/product/apis-integrations/rest-api#configuration-base-path)。
