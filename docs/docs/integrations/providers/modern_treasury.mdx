# Modern Treasury

>[Modern Treasury](https://www.moderntreasury.com/) 简化了复杂的支付操作。它是一个统一的平台，用于支持资金流动的业务和流程。
>- 连接银行和支付系统
>- 实时跟踪交易和余额
>- 自动化支付操作以实现规模化
