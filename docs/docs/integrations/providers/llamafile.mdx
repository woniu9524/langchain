# llamafile

>[llamafile](https://github.com/Mozilla-Ocho/llamafile) 允许您通过单个文件分发和运行 LLM。

>`llamafile` 使开源 LLM 对开发人员和最终用户都更加易于访问。
>`llamafile` 通过将 [llama.cpp](https://github.com/ggerganov/llama.cpp) 与
>[Cosmopolitan Libc](https://github.com/jart/cosmopolitan) 结合到一个框架中，将 LLM 的所有复杂性
>压缩为单个文件可执行文件（称为“llamafile”），该文件无需安装即可在大多数计算机上本地运行。


## 安装和设置

请参阅[安装说明](https://github.com/Mozilla-Ocho/llamafile?tab=readme-ov-file#quickstart)。

请参阅[用法示例](/docs/integrations/llms/llamafile)。

