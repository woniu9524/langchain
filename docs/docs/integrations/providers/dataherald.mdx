# Dataherald

>[Dataherald](https://www.dataherald.com) 是一个自然语言到 SQL 的工具。

本文档介绍了如何在 LangChain 中使用 `Dataherald API`。

## 安装和设置
- 使用以下命令安装所需依赖：
```bash
pip install dataherald
```
- 前往 dataherald [此处](https://www.dataherald.com) 注册。
- 创建一个应用并获取您的 `API KEY`。
- 将您的 `API KEY` 设置为环境变量 `DATAHERALD_API_KEY`。

## 封装器

### 工具类

存在一个名为 `DataheraldAPIWrapper` 的工具类，用于封装此 API。导入该工具类：

```python
from langchain_community.utilities.dataherald import DataheraldAPIWrapper
```

有关此工具类的更详细教程，请参阅 [此 Notebook](/docs/integrations/tools/dataherald)。

### 工具

您可以在 Agent 中像这样使用该工具：
```python
from langchain_community.utilities.dataherald import DataheraldAPIWrapper
from langchain_community.tools.dataherald.tool import DataheraldTextToSQL
from langchain_openai import ChatOpenAI
from langchain import hub
from langchain.agents import AgentExecutor, create_react_agent, load_tools

api_wrapper = DataheraldAPIWrapper(db_connection_id="<db_connection_id>")
tool = DataheraldTextToSQL(api_wrapper=api_wrapper)
llm = ChatOpenAI(model="gpt-3.5-turbo", temperature=0)
prompt = hub.pull("hwchase17/react")
agent = create_react_agent(llm, tools, prompt)
agent_executor = AgentExecutor(agent=agent, tools=tools, verbose=True, handle_parsing_errors=True)
agent_executor.invoke({"input":"Return the sql for this question: How many employees are in the company?"})
```
