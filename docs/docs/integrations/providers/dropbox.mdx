# Dropbox

>[Dropbox](https://en.wikipedia.org/wiki/Dropbox) 是一个文件托管服务，可以将一切——传统文件、云内容和网站快捷方式——整合到同一个地方。
 

## 安装与设置

请参阅详细的[安装指南](/docs/integrations/document_loaders/dropbox#prerequisites)。

```bash
pip install -U dropbox
```

## 文档加载器

请参阅[用法示例](/docs/integrations/document_loaders/dropbox)。

```python
from langchain_community.document_loaders import DropboxLoader
```
