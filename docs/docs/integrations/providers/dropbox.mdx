# Dropbox

>[Dropbox](https://en.wikipedia.org/wiki/Dropbox) 是一个文件托管服务，可将所有内容——传统文件、云内容和 Web 快捷方式——整合到一个地方。
 

## 安装和设置

请参阅详细的[安装指南](/docs/integrations/document_loaders/dropbox#prerequisites)。

```bash
pip install -U dropbox
```

from langchain_community.document_loaders import DropboxLoader
