# Everly AI

> [Everly AI](https://everlyai.xyz/) 允许您在云端大规模运行机器学习模型。
> 它还提供对[多个 LLM 模型](https://everlyai.xyz/)的 API 访问。

## 安装和设置

要使用 `Everly AI`，您需要一个 API 密钥。请访问
[Everly AI](https://everlyai.xyz/) 在您的个人资料中创建 API 密钥。
