# Beautiful Soup

>[Beautiful Soup](https://www.crummy.com/software/BeautifulSoup/) 是一个用于解析 HTML 和 XML 文档的 Python 包（包括格式不正确的标记，即未闭合的标签，因此得名 tag soup）。它为解析后的页面创建了一个解析树，可用于从 HTML 中提取数据，这对于网页抓取非常有用。

## 安装和设置

```bash
pip install beautifulsoup4
```
