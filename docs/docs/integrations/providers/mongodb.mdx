# MongoDB

>[MongoDB](https://www.mongodb.com/) 是一个 NoSQL 文档数据库，支持类似 JSON 的文档，具有动态模式。

**注意：**
- 请参阅 [MongoDB Atlas 页面](/docs/integrations/providers/mongodb_atlas) 上其他 `MongoDB` 集成。

## 安装与设置

安装 Python 包：

```bash
pip install langchain-mongodb
```

## 消息历史记录

请参阅[用法示例](/docs/integrations/memory/mongodb_chat_message_history)。

导入此向量存储：
```python
from langchain_mongodb.chat_message_histories import MongoDBChatMessageHistory
```
