# Memgraph

>Memgraph 是一个高性能的内存图数据库，针对实时查询和分析进行了优化。
>访问[他们的网站](https://memgraph.com/)即可开始使用 Memgraph。

## 安装与设置

- 使用 `pip install langchain-memgraph` 安装 Python SDK

## MemgraphQAChain

Memgraph 图数据库提供了一个包装器，允许您根据用户输入生成 Cypher 语句，并使用这些语句从数据库中检索相关信息。

```python
from langchain_memgraph.chains.graph_qa import MemgraphQAChain
from langchain_memgraph.graphs.memgraph import MemgraphLangChain
```

请参阅[使用示例](/docs/integrations/graphs/memgraph)

## 从非结构化数据构建知识图谱

您可以使用此集成从非结构化数据构建知识图谱。

```python
from langchain_memgraph.graphs.memgraph import MemgraphLangChain
from langchain_experimental.graph_transformers import LLMGraphTransformer
```

请参阅[使用示例](/docs/integrations/graphs/memgraph)
