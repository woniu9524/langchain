> `Groq LPU` 拥有确定性的单核流式处理架构，为生成式 AI 推理速度树立了标准，可为任何给定的工作负载提供可预测且可重复的性能。
>
>除了架构之外，`Groq` 软件还旨在赋能广大开发者，提供所需的工具来创建创新、强大的 AI 应用。
>
>以 Groq 作为您的引擎，您可以：
>* 实现无损耗的低延迟和高性能，用于实时 AI 和 HPC 推理 🔥
>* 了解任何给定工作负载的确切性能和计算时间 🔮
>* 利用我们的尖端技术保持竞争优势 💪

## 安装和设置

安装集成包：

```bash
pip install langchain-groq
```

申请一个 [API 密钥](https://console.groq.com/login?utm_source=langchain&utm_content=provider_page) 并将其设置为环境变量：

```bash
export GROQ_API_KEY=gsk_...
```
