## Chat models

请参阅[安装说明](/docs/integrations/chat/yuan2/#setting-up-your-api-server)。

Yuan2.0 提供了一个与 OpenAI 兼容的 API，而 ChatYuan2 通过使用 `OpenAI client` 集成到了 langchain 中。
因此，请确保已安装 `openai` 包。

```bash
pip install openai
```
请参阅[使用示例](/docs/integrations/chat/yuan2)。

```python
from langchain_community.chat_models import ChatYuan2
```
