# Anyscale

>[Anyscale](https://www.anyscale.com) 是一个通过生产级 API 来运行、微调和扩展 LLM 的平台。
> [Anyscale Endpoints](https://docs.anyscale.com/endpoints/overview) 以经济高效的方式提供许多开源模型。

`Anyscale` 还提供了一个关于如何设置 `LangChain` 与 `Anyscale` 集成以构建高级聊天代理的[示例](https://docs.anyscale.com/endpoints/model-serving/examples/langchain-integration)。

## 安装和设置

- 获取 Anyscale 服务 URL、路由和 API 密钥，并将它们设置为环境变量（`ANYSCALE_SERVICE_URL`、`ANYSCALE_SERVICE_ROUTE`、`ANYSCALE_SERVICE_TOKEN`）。
- 有关更多详细信息，请参阅 [Anyscale 文档](https://www.anyscale.com/get-started)。

我们需要安装 `openai` 包：

```bash
pip install openai
```

## LLM

请参阅[使用示例](/docs/integrations/llms/anyscale)。
