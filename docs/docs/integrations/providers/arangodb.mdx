# ArangoDB

>[ArangoDB](https://github.com/arangodb/arangodb) 是一个可扩展的图数据库系统，可更快地从连接的数据中提取价值。通过单一查询语言，支持原生图、集成搜索引擎和 JSON。ArangoDB 可在本地、云端或任何地方运行。

## 安装和设置

使用以下命令安装 [ArangoDB Python Driver](https://github.com/ArangoDB-Community/python-arango) 包：

```bash
pip install python-arango
```
