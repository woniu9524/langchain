# ArangoDB

>[ArangoDB](https://github.com/arangodb/arangodb) 是一个可扩展的图数据库系统，可通过单一查询语言，利用本地图、集成搜索引擎和 JSON 支持，更快地从连接数据中获取价值。ArangoDB 可在本地、云端——任何地方运行。

## 安装与设置

使用以下命令安装 [ArangoDB Python Driver](https://github.com/ArangoDB-Community/python-arango) 包：

```bash
pip install python-arango
```

## 图 QA 链

将您的 `ArangoDB` 数据库与聊天模型连接，以获得数据洞察。

请在此处查看笔记本示例：[/docs/integrations/graphs/arangodb](/docs/integrations/graphs/arangodb)。

```python
from arango import ArangoClient

from langchain_community.graphs import ArangoGraph
from langchain.chains import ArangoGraphQAChain
```
