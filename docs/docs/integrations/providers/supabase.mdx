# Supabase (Postgres)

>[Supabase](https://supabase.com/docs) 是一个开源的 `Firebase` 替代方案。
> `Supabase` 构建在 `PostgreSQL` 之上，它提供了强大的 `SQL` 查询能力，并能够与现有的工具和框架实现简单的接口对接。

>[PostgreSQL](https://en.wikipedia.org/wiki/PostgreSQL) 也被称为 `Postgres`，
> 是一个免费且开源的关系数据库管理系统 (RDBMS)，它注重可扩展性和 `SQL` 合规性。

## 安装和设置

我们需要安装 `supabase` Python 包。

```bash
pip install supabase
```
