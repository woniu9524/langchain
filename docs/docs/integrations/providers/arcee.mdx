# Arcee

>[Arcee](https://www.arcee.ai/about/about-us) 赋能我们称之为 SLM（小型、专业化、安全且可扩展的语言模型）的开发和进步。通过提供 SLM 适配系统和无缝、安全的集成，`Arcee` 使企业能够充分发挥领域适应性语言模型的潜力，驱动运营的变革性创新。


## 安装与设置

获取你的 `Arcee API` 密钥。


## LLMs

请参阅[使用示例](/docs/integrations/llms/arcee)。

```python
from langchain_community.llms import Arcee
```
