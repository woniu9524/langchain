# Arcee

>[Arcee](https://www.arcee.ai/about/about-us) 赋能我们称之为 SLM（小型、专业化、安全且可扩展的语言模型）的开发和进步。
>通过提供 SLM 适配系统和无缝、安全的集成，
>`Arcee` 使企业能够充分发挥领域适配语言模型的潜力，
>推动运营的变革性创新。

## 安装和设置

获取您的 `Arcee API` 密钥。

## LLM

请参阅 [用法示例](/docs/integrations/llms/arcee)。

```python
from langchain_community.llms import Arcee
```
