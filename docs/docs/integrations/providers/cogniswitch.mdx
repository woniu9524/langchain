# CogniSwitch

>[CogniSwitch](https://www.cogniswitch.ai/aboutus) 是一个基于 API 的数据平台，
>通过提取实体、概念及其关系来增强企业数据，
>从而将数据转换为多维格式并存储在能够适应这些增强的数据的数据库中。
>在我们的案例中，数据存储在知识图中。
>这些增强后的数据现在已准备好供 LLM 和其他 GenAI 应用程序使用，
>确保数据可消耗且能维护上下文。
>从而消除幻觉并提供准确性。

## Toolkit

请参阅[安装说明和用法示例](/docs/integrations/tools/cogniswitch)。

from langchain_community.agent_toolkits import CogniswitchToolkit
