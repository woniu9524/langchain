# Transwarp

>[Transwarp](https://www.transwarp.cn/en/introduction) 旨在构建企业级大数据和人工智能基础设施软件，塑造数据世界的未来。它围绕数据全生命周期，包括集成、存储、治理、建模、分析、挖掘和流通，为企业提供基础设施软件和服务。

`Transwarp` 专注于技术研发，并在以下方面积累了核心技术：分布式计算、SQL 编译、数据库技术、多模型数据管理统一、基于容器的云计算以及大数据分析与智能。

## 安装

您需要安装几个 Python 包：

```bash
pip install -U tiktoken hippo-api
```

并获取连接配置。
