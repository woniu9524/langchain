# Yeager.ai

此页面介绍了如何使用 [Yeager.ai](https://yeager.ai) 来生成 LangChain 工具和代理。

## 什么是 Yeager.ai？
Yeager.ai 是一个旨在简化 AI 工具和代理创建过程的生态系统。

它提供了 yAgents，一个无代码 LangChain 代理构建器，使用户能够轻松地构建、测试和部署 AI 解决方案。yAgents 利用 LangChain 框架，可以与各种语言模型和资源无缝集成，使其适用于开发人员、研究人员以及跨不同应用的 AI 爱好者。

## yAgents
一个低代码生成代理，旨在帮助您轻松构建、原型设计和部署 Langchain 工具。

### 如何使用？
```
pip install yeagerai-agent
yeagerai-agent
```
访问 http://127.0.0.1:7860

这将安装必要的依赖项并在您的系统上设置 yAgents。首次运行后，yAgents 将创建一个 .env 文件，您可以在其中输入您的 OpenAI API 密钥。您也可以直接从 Gradio 界面上的“Settings”选项卡进行操作。

`OPENAI_API_KEY=<your_openai_api_key_here>`

我们推荐使用 GPT-4。然而，如果问题被充分分解，该工具也可以与 GPT-3 一起使用。

### 使用 yAgents 创建和执行工具
yAgents 使创建和执行 AI 驱动的工具变得容易。以下是该过程的简要概述：
1. 创建一个工具：要创建一个工具，请向 yAgents 提供一个自然语言提示。提示应清楚地描述工具的目的和功能。例如：
`create a tool that returns the n-th prime number`

2. 将工具加载到工具集中：要将工具加载到 yAgents 中，只需向 yAgents 提供一个指令即可。例如：
`load the tool that you just created it into your toolkit`

3. 执行工具：要运行工具或代理，只需向 yAgents 提供一个包含工具名称和任何必需参数的命令。例如：
`generate the 50th prime number`
