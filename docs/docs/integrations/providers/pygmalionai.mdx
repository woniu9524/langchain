# PygmalionAI

>[PygmalionAI](https://pygmalion.chat/) 是一家支持开源模型 Thus，通过为 [Aphrodite Engine](https://github.com/PygmalionAI/aphrodite-engine) 提供推理端点来提供服务。


## 安装和设置


```bash
pip install aphrodite-engine
```
