# PygmalionAI

>[PygmalionAI](https://pygmalion.chat/) 是一家公司，通过为 [Aphrodite Engine](https://github.com/PygmalionAI/aphrodite-engine) 提供推理端点来支持开源模型。


## 安装和设置


```bash
pip install aphrodite-engine
```
