# Tigris

> [Tigris](https://tigrisdata.com) 是一个开源的无服务器 NoSQL 数据库和搜索平台，旨在简化高性能向量搜索应用程序的构建。
> `Tigris` 消除了管理、操作和同步多个工具的基础设施复杂性，让你能够专注于构建出色的应用程序。

## Installation and Setup


```bash
pip install tigrisdb openapi-schema-pydantic
```
