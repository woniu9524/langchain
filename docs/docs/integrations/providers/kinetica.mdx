# Kinetica

[Kinetica](https://www.kinetica.com/) 是一个专为时间序列和空间数据上的分析和生成式 AI 而设计的实时数据库。

## Chat Model

Kinetica LLM 包装器使用 [Kinetica SqlAssist LLM](https://docs.kinetica.com/7.2/sql-gpt/concepts/) 将自然语言转换为 SQL，以简化数据检索过程。

有关用法，请参阅 [Kinetica Language To SQL Chat Model](/docs/integrations/chat/kinetica)。

```python
from langchain_community.chat_models.kinetica import ChatKinetica
```

## Vector Store

Kinetca vectorstore 包装器利用 Kinetica 对 [向量相似性搜索](https://docs.kinetica.com/7.2/vector_search/) 的原生支持。

有关用法，请参阅 [Kinetica Vectorstore API](/docs/integrations/vectorstores/kinetica)。

```python
from langchain_community.vectorstores import Kinetica
```

## Document Loader

Kinetica Document loader 可用于从 [Kinetica](https://www.kinetica.com/) 数据库加载 LangChain [Documents](https://python.langchain.com/api_reference/core/documents/langchain_core.documents.base.Document.html)。

有关用法，请参阅 [Kinetica Document Loader](/docs/integrations/document_loaders/kinetica)。

```python
from langchain_community.document_loaders.kinetica_loader import KineticaLoader
```

## Retriever

Kinetica Retriever 可以根据非结构化查询返回文档。

有关用法，请参阅 [Kinetica VectorStore based Retriever](/docs/integrations/retrievers/kinetica)。
