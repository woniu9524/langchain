# 百度

>[百度云](https://cloud.baidu.com/) 是由总部位于北京的`百度公司`提供的云服务。它提供云存储服务、客户端软件、文件管理、资源共享和第三方集成。

## 安装与设置

在此处注册并获取`Qianfan`的`AK`和`SK`密钥：[https://cloud.baidu.com/product/wenxinworkshop](https://cloud.baidu.com/product/wenxinworkshop)。

## LLMs

### 百度千帆

请参阅[使用示例](/docs/integrations/llms/baidu_qianfan_endpoint)。
