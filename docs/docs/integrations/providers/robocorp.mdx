# Sema4 (原 Robocorp)

>[ Robocorp ](https://robocorp.com/) 帮助构建和操作无缝运行、随时随地、任意规模的 Python 机器人。


## 安装和设置

您需要安装 `langchain-robocorp` Python 包：

```bash
pip install langchain-robocorp
```

您需要一个正在运行的 `Action Server` 实例才能从您的代理应用程序进行通信。
请参阅 [Robocorp Quickstart](https://github.com/robocorp/robocorp#quickstart) 以了解如何设置 Action Server 和创建您的 Actions。

您可以使用 Action Server 的 `new` 命令来引导一个新项目。

```bash
action-server new
cd ./your-project-name
action-server start
```
