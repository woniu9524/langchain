# Sema4 (原 Robocorp)

>[Robocorp](https://robocorp.com/) 帮助构建和运行无缝运行、可任意扩展的 Python worker。

## 安装和设置

您需要安装 `langchain-robocorp` Python 包：

```bash
pip install langchain-robocorp
```

您需要一个正在运行的 `Action Server` 实例，以便从您的 agent 应用程序与其通信。
有关如何设置 Action Server 和创建 Actions 的信息，请参阅 [Robocorp Quickstart](https://github.com/robocorp/robocorp#quickstart)。

您可以使用 Action Server 的 `new` 命令来引导一个新项目。

```bash
action-server new
cd ./your-project-name
action-server start
```
