# Cognee

Cognee 实现了可扩展、模块化的 ECL（提取、认知、加载）管道，让您能够连接和检索过去的对话、文档和音频转录，同时减少幻觉、开发工作量和成本。

Cognee 融合了图数据库和向量数据库，以揭示数据中隐藏的关系和新模式。您可以自动建模、加载和检索代表您的业务领域实体和对象，并分析它们的关系，从而揭示单独的向量存储或图存储无法提供的见解。
