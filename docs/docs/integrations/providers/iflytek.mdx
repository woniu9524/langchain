# iFlytek

>[iFlytek](https://www.iflytek.com) 是一家成立于1999年的中国信息技术公司，
>它创建了语音识别软件和基于语音的互联网/移动产品，涵盖教育、通信、
>音乐、智能玩具等行业。


## 安装和设置

- 从 [iFlyTek SparkLLM API Console](https://console.xfyun.cn/services/bm3) 获取 `SparkLLM` 的 app_id、api_key 和 api_secret（更多信息请参阅 [iFlyTek SparkLLM 简介](https://xinghuo.xfyun.cn/sparkapi)）。
- 安装 Python 包（不适用于 embedding 模型）：

```bash
pip install websocket-client
```
