# LLMs

:::caution
您当前所在页面记录的是 [文本补全模型](/docs/concepts/text_llms) 的使用方法。许多最新且最受欢迎的模型都是 [聊天补全模型](/docs/concepts/chat_models)。

除非您专门使用更高级的提示技术，否则您可能在寻找[这个页面](/docs/integrations/chat/)。
:::

[LLMs](/docs/concepts/text_llms) 是语言模型，它们接受字符串作为输入，并返回字符串作为输出。

:::info

如果您想自己编写 LLM，请参阅[此操作指南](/docs/how_to/custom_llm/)。
如果您想贡献集成，请参阅[贡献集成方法](/docs/contributing/how_to/integrations/)。

:::
