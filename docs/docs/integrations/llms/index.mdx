# LLMs

:::caution
您目前正在浏览关于[文本补全模型](/docs/concepts/text_llms)使用方法的文档。许多最新和最受欢迎的模型是[聊天补全模型](/docs/concepts/chat_models)。

除非您专门使用更高级的提示技术，否则您可能在寻找[此页面](/docs/integrations/chat/)。
:::

[LLMs](/docs/concepts/text_llms) 是接收字符串作为输入并返回字符串作为输出的语言模型。

:::info

如果您想编写自己的 LLM，请参阅[此操作指南](/docs/how_to/custom_llm/)。
如果您想贡献集成，请参阅[贡献集成](/docs/contributing/how_to/integrations/)。

:::
