# Chat models

[Chat models](/docs/concepts/chat_models) 是使用一系列 [messages](/docs/concepts/messages) 作为输入并返回 messages 作为输出的语言模型（而不是使用纯文本）。这些通常是较新的模型。

:::info

如果您想编写自己的 chat model，请参阅[此指南](/docs/how_to/custom_chat_model/)。
如果您想贡献一个集成，请参阅[贡献集成](/docs/contributing/how_to/integrations/)。

:::
