# 检索器

[检索器](/docs/concepts/retrievers)是一种接口，它根据非结构化查询返回文档。
它比向量存储更通用。
检索器不需要能够存储文档，只需要能够返回（或检索）它们。
检索器可以从向量存储创建，但其范围也足够广泛，包括[维基百科搜索](/docs/integrations/retrievers/wikipedia/)和[Amazon Kendra](/docs/integrations/retrievers/amazon_kendra_retriever/)。

检索器接受字符串查询作为输入，并返回一个 [Documents](https://python.langchain.com/api_reference/core/documents/langchain_core.documents.base.Document.html) 列表作为输出。

有关如何使用检索器的具体信息，请参阅[此处相关的操作指南](/docs/how_to/#retrievers)。

请注意，所有[向量存储](/docs/concepts/vectorstores)都可以[转换为检索器](/docs/how_to/vectorstore_retriever/)。
有关可用的向量存储，请参阅向量存储[集成文档](/docs/integrations/vectorstores/)。
本页列出了通过继承[BaseRetriever](/docs/how_to/custom_retriever/)实现的自定义检索器。
