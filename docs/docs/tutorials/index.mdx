# 教程

刚开始接触 LangChain 或通用的 LLM 应用开发？阅读本材料，快速上手构建你的第一个应用程序。

## 入门

通过构建简单的应用程序，熟悉 LangChain 的开源组件。

如果你想从特定的提供商开始使用 [聊天模型](/docs/integrations/chat/)、[向量存储](/docs/integrations/vectorstores/)
或其他 LangChain 组件，请查看我们支持的 [集成](/docs/integrations/providers/)。

- [聊天模型和提示](/docs/tutorials/llm_chain)：使用 [提示模板](/docs/concepts/prompt_templates) 和 [聊天模型](/docs/concepts/chat_models) 构建简单的 LLM 应用程序。
- [语义搜索](/docs/tutorials/retrievers)：使用 [文档加载程序](/docs/concepts/document_loaders)、[嵌入模型](/docs/concepts/embedding_models/) 和 [向量存储](/docs/concepts/vectorstores/) 构建基于 PDF 的语义搜索引擎。
- [分类](/docs/tutorials/classification)：使用带有 [结构化输出](/docs/concepts/structured_outputs/) 的 [聊天模型](/docs/concepts/chat_models) 将文本分类到不同的类别或标签。
- [提取](/docs/tutorials/extraction)：使用 [聊天模型](/docs/concepts/chat_models) 和 [少样本示例](/docs/concepts/few_shot_prompting/) 从文本和其他非结构化媒体中提取结构化数据。

有关使用所有 LangChain 组件的更多详细信息，请参阅 [操作指南](/docs/how_to)。

## 编排

开始使用 [LangGraph](https://langchain-ai.github.io/langgraph/) 将 LangChain 组件组合成功能齐全的应用程序。

- [聊天机器人](/docs/tutorials/chatbot)：构建一个集成记忆的聊天机器人。
- [代理](/docs/tutorials/agents)：构建一个与外部工具交互的代理。
- [检索增强生成 (RAG) 第一部分](/docs/tutorials/rag)：构建一个利用你自己的文档来指导其响应的应用程序。
- [检索增强生成 (RAG) 第二部分](/docs/tutorials/qa_chat_history)：构建一个集成用户交互记忆和多步检索的 RAG 应用程序。
- [SQL 问答](/docs/tutorials/sql_qa)：构建一个执行 SQL 查询来指导其响应的问答系统。
- [摘要](/docs/tutorials/summarization)：为（可能很长的）文本生成摘要。
- [图数据库问答](/docs/tutorials/graph)：构建一个查询图数据库来指导其响应的问答系统。

## LangSmith

LangSmith 允许你密切跟踪、监控和评估你的 LLM 应用程序。
它与 LangChain 无缝集成，你可以使用它来检查和调试你构建时链的各个步骤。

LangSmith 文档托管在单独的网站上。
你可以在此处浏览 [LangSmith 教程](https://docs.smith.langchain.com/)。
